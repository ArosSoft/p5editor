# Шаг 1: Переменные и последовательность действий

## Что такое переменная?

**Переменная** — это «коробочка» для хранения данных. Ты можешь положить в неё число, текст или другое значение, а потом использовать его в программе.

## Объявление переменных в JavaScript

В JavaScript есть несколько способов объявить переменную:

```javascript
let x = 5;        // Изменяемая переменная
const PI = 3.14;  // Константа (нельзя изменить)
var old = 10;     // Старый способ (лучше не использовать)
```

## Пример: вычисление суммы

```javascript
let x = 5;
let y = 10;
let sum = x + y;
let message = `Сумма ${x} + ${y} = ${sum}`;
console.log(message); // Вывод: "Сумма 5 + 10 = 15"
```

## Как это работает?

1. `let x = 5;` — создаём переменную `x` и кладём в неё число 5
2. `let y = 10;` — создаём переменную `y` и кладём в неё число 10
3. `let sum = x + y;` — складываем значения переменных и сохраняем результат
4. `console.log()` — выводим сообщение в консоль
