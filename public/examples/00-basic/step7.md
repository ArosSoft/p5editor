# Шаг 7: Сортировка пузырьком

## Что такое сортировка?

**Сортировка** — это упорядочивание элементов массива по возрастанию или убыванию. Это одна из самых важных задач в программировании.

## Сортировка пузырьком (Bubble Sort)

Алгоритм многократно проходит по массиву, сравнивая соседние элементы и меняя их местами, если они стоят в неправильном порядке.

## Пример кода

```javascript
let arr = [64, 34, 25, 12, 22, 11, 90];

for (let i = 0; i < arr.length; i++) {
  for (let j = 0; j < arr.length - i - 1; j++) {
    if (arr[j] > arr[j + 1]) {
      // Меняем элементы местами
      [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
    }
  }
}

console.log("Отсортировано:", arr);
// [11, 12, 22, 25, 34, 64, 90]
```

## Как это работает?

```
Исходный: [64, 34, 25, 12, 22, 11, 90]

Первый проход:
[34, 64, 25, 12, 22, 11, 90]  (64 > 34, меняем)
[34, 25, 64, 12, 22, 11, 90]  (64 > 25, меняем)
[34, 25, 12, 64, 22, 11, 90]  (64 > 12, меняем)
[34, 25, 12, 22, 64, 11, 90]  (64 > 22, меняем)
[34, 25, 12, 22, 11, 64, 90]  (64 > 11, меняем)
[34, 25, 12, 22, 11, 64, 90]  (64 < 90, не меняем)

Повторяем, пока массив не отсортируется
```

## Классическая замена через временную переменную

```javascript
if (arr[j] > arr[j + 1]) {
  let temp = arr[j];
  arr[j] = arr[j + 1];
  arr[j + 1] = temp;
}
```

## Сортировка по убыванию

```javascript
let arr = [64, 34, 25, 12, 22];

for (let i = 0; i < arr.length; i++) {
  for (let j = 0; j < arr.length - i - 1; j++) {
    if (arr[j] < arr[j + 1]) { // Меняем знак на <
      [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
    }
  }
}

console.log("По убыванию:", arr); // [64, 34, 25, 22, 12]
```

## Встроенная сортировка JavaScript

```javascript
let numbers = [64, 34, 25, 12, 22];

// Сортировка чисел (обязательно с компаратором!)
numbers.sort((a, b) => a - b);
console.log(numbers); // [12, 22, 25, 34, 64]

// Сортировка строк (по алфавиту)
let names = ["Олег", "Анна", "Борис"];
names.sort();
console.log(names); // ["Анна", "Борис", "Олег"]
```
