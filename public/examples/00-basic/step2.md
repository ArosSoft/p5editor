# Шаг 2: Условные операторы (if-else)

## Что такое условные операторы?

**Условные операторы** позволяют программе принимать решения. Они выполняют разный код в зависимости от того, истинно условие или ложно.

## Синтаксис if-else

```javascript
if (условие) {
  // Выполняется, если условие ИСТИННО
} else {
  // Выполняется, если условие ЛОЖНО
}
```

## Пример: проверка на чётность

```javascript
let number = parseInt(prompt("Введи число:"));

if (number % 2 === 0) {
  console.log(`${number} — чётное!`);
} else {
  console.log(`${number} — нечётное!`);
}
```

## Операторы сравнения

| Оператор | Значение | Пример |
|----------|----------|--------|
| `===` | Равно | `5 === 5` → true |
| `!==` | Не равно | `5 !== 3` → true |
| `>` | Больше | `10 > 5` → true |
| `<` | Меньше | `3 < 7` → true |
| `>=` | Больше или равно | `5 >= 5` → true |
| `<=` | Меньше или равно | `3 <= 7` → true |

## Логические операторы

```javascript
// И (&&) — оба условия должны быть истинны
if (age >= 18 && hasLicense) {
  console.log("Можно водить машину");
}

// ИЛИ (||) — хотя бы одно условие истинно
if (day === "суббота" || day === "воскресенье") {
  console.log("Выходной!");
}

// НЕ (!) — инвертирует значение
if (!isRaining) {
  console.log("Можно идти гулять");
}
```
