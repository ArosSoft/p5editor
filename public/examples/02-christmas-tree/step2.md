# Шаг 2: Рисуем зелёную ёлку 🌲

## Продолжаем рисовать

Теперь самое главное — нарисуем саму ёлку! Она будет состоять из трёх треугольников.

## Рисуем ярусы ёлки

### 1. Нижний ярус

```javascript
fill(0, 100, 0);  // Тёмно-зелёный цвет
triangle(50, 250, 350, 250, 200, 120);
```

**`triangle()`** — рисует треугольник. В скобках три пары координат (x, y) для каждой вершины:
- Первая точка: (50, 250) — левый нижний угол
- Вторая точка: (350, 250) — правый нижний угол  
- Третья точка: (200, 120) — верхушка треугольника

![картинка1](images/221.png)

### 2. Средний ярус

```javascript
fill(0, 120, 0);  // Чуть светлее
triangle(80, 190, 320, 190, 200, 80);
```

Обрати внимание: этот треугольник меньше и расположен выше. Цвет тоже немного светлее — так ёлка выглядит объёмнее!

![картинка2](images/222.png)

### 3. Верхний ярус

```javascript
fill(0, 140, 0);  // Ещё светлее
triangle(110, 140, 290, 140, 200, 40);
```

Самый маленький и светлый треугольник наверху!

![картинка3](images/223.png)

### 4. Рисуем ствол

Ёлка не может стоять без ствола!

```javascript
fill(101, 67, 33);  // Коричневый цвет
rect(180, 240, 40, 80);  // Прямоугольник
```

**`rect()`** — рисует прямоугольник. В скобках: (x, y, ширина, высота).

![картинка4](images/224.png)
