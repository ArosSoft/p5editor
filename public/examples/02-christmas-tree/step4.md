# Шаг 4: Поиграй! Добавляем игрушки мышкой 🎮

## Интерактивная ёлка

Самое интересное — теперь мы можем добавлять игрушки на ёлку, просто кликая мышкой!

## Как это работает?

### Волшебная функция `mousePressed()`

```javascript
function mousePressed() {
  // Этот код выполняется каждый раз, когда ты кликаешь мышкой
}
```

> 📷 **[МЕСТО ДЛЯ ИЗОБРАЖЕНИЯ: скриншот ёлки с несколькими игрушками]**

### Проверяем, куда кликнули

Нам нужно убедиться, что игрушка появится именно на ёлке, а не в другом месте:

```javascript
let c = get(mouseX, mouseY);  // Узнаём цвет пикселя под курсором
```

**`get()`** — как пипетка, «берёт» цвет из нужной точки.  
**`mouseX`** и **`mouseY`** — координаты курсора мыши.

### Проверяем цвет

```javascript
if (c[1] > 100 && c[0] < 100 && c[2] < 100) {
  // Если цвет зелёный — добавляем игрушку!
}
```

Это условие проверяет: много ли зелёного цвета (c[1]) и мало ли красного (c[0]) и синего (c[2]). Если да — значит, мы кликнули на ёлку!

> 📷 **[МЕСТО ДЛЯ ИЗОБРАЖЕНИЯ: схема проверки цвета пикселя]**

### Рисуем игрушку

```javascript
fill(random(200, 255), random(0, 100), random(0, 100));  // Случайный цвет
noStroke();
circle(mouseX, mouseY, 15);  // Круглая игрушка диаметром 15 пикселей
```

**`random()`** — выбирает случайное число в указанном диапазоне. Так каждая игрушка будет уникального цвета!

### Добавляем блик

Чтобы игрушка выглядела объёмной, добавим маленький светлый блик:

```javascript
fill(255, 255, 255, 150);  // Полупрозрачный белый
circle(mouseX - 3, mouseY - 3, 4);  // Блик слева-сверху
```

> 📷 **[МЕСТО ДЛЯ ИЗОБРАЖЕНИЯ: крупный план игрушки с бликом]**
