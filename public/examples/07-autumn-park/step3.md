# Урок 3: Создаём траву и листья

Теперь наполним парк травой и листьями. Трава будет расти на земле, а листья — на деревьях (потом они будут падать!).

## Создаём травинки
Трава растёт вдоль всего низа холста. Используем цикл, чтобы создать много травинок:

```javascript
  // Создаём травинки
  for (let x = 0; x < width; x += 4) {
    grassBlades.push({
      x: x + random(-2, 2),
      height: random(8, 20),
      tilt: random(-4, 4)
    });
  }
```

## Что делает `x += 4`?
Это значит `x = x + 4`. Цикл начинается с 0 и увеличивает `x` на 4 каждый раз. Так мы создаём травинки каждые 4 пикселя — получается около 100 травинок!

## Свойства травинки
*   `x` — позиция по горизонтали (с небольшим случайным сдвигом)
*   `height` — высота травинки (от 8 до 20 пикселей)
*   `tilt` — наклон в сторону (от -4 до 4 пикселей)

## Создаём листья на деревьях
Добавь этот код после цикла с травой:

```javascript
  // Создаём листья на деревьях
  for (let i = 0; i < 60; i++) {
    leaves.push(createLeaf());
  }
```

## Что такое `createLeaf()`?
Это **функция** — отдельный блок кода, который можно вызывать по имени. Мы напишем эту функцию позже. Сейчас просто запомним: она создаёт один листик со случайными свойствами.

![Трава и листья схема](/images/grass-leaves-diagram.png)

### Почему 60 листьев?
Программисты подбирают число так, чтобы было красиво и компьютер не тормозил. 60 листьев — достаточно для густого листопада, но не слишком много для производительности.

### Задание
Попробуй изменить количество:
1.  Сделай траву реже: измени `x += 4` на `x += 8`
2.  Уменьши количество листьев: поставь `i < 30`
