# Урок 4: Рисуем небо, солнце и холмы

Перед тем как рисовать деревья, создадим фон осеннего парка: небо с облаками, тёплое солнце и холмы на заднем плане.

## Функция draw()
Всё, что мы нарисуем в `draw()`, появится на экране. Эта функция выполняется много раз в секунду (60 кадров!).

Добавь этот код в функцию `draw()`:

```javascript
function draw() {
  // Реалистичное небо с облаками
  drawSky();

  // Солнце с ореолом
  drawSun();

  // Дальний план (холмы)
  drawBackgroundHills();

  // Земля
  fill(34, 89, 34);
  rect(0, 280, width, 120);
}
```

## Что такое вызов функции?
Мы пишем имя функции со скобками: `drawSky()`. Компьютер переходит к этой функции, выполняет весь код внутри и возвращается обратно.

## Рисуем небо с градиентом
Функция `drawSky()` рисует небо с плавным переходом цвета и облаками:

```javascript
function drawSky() {
  // Градиент неба
  for (let i = 0; i <= height; i++) {
    let inter = map(i, 0, height * 0.6, 0, 1);
    let c = lerpColor(color(100, 150, 230), color(255, 200, 150), inter);
    stroke(c);
    line(0, i, width, i);
  }

  // Облака
  noStroke();
  fill(255, 255, 255, 200);
  ellipse(50, 50, 60, 30);
  ellipse(80, 40, 50, 25);
  ellipse(300, 80, 80, 35);
  ellipse(340, 70, 60, 25);
}
```

## Что делает `lerpColor`?
Функция `lerpColor()` плавно смешивает два цвета:
*   `color(100, 150, 230)` — голубой (верх неба)
*   `color(255, 200, 150)` — оранжевый (горизонт)
*   `inter` — число от 0 до 1, показывающее, насколько смешать цвета

`map(i, 0, height * 0.6, 0, 1)` переводит позицию `i` из диапазона высот в число от 0 до 1.

## Рисуем солнце с ореолом
Функция `drawSun()` создаёт солнце с мягким свечением вокруг:

```javascript
function drawSun() {
  // Ореол
  noStroke();
  for (let i = 3; i > 0; i--) {
    fill(255, 200, 100, 30 - i * 5);
    ellipse(350, 70, 70 + i * 15, 70 + i * 15);
  }

  // Солнце
  fill(255, 220, 100);
  ellipse(350, 70, 60, 60);
}
```

## Как работает ореол?
Цикл рисует 3 полупрозрачных круга вокруг солнца:
*   `i = 3` — самый большой и прозрачный
*   `i = 2` — средний
*   `i = 1` — маленький и менее прозрачный

`30 - i * 5` вычисляет прозрачность: чем больше `i`, тем прозрачнее круг.

## Рисуем холмы на заднем плане
Холмы создают глубину сцены:

```javascript
function drawBackgroundHills() {
  // Холмы на заднем плане
  noStroke();
  fill(60, 120, 60, 100);
  beginShape();
  vertex(0, 200);
  vertex(100, 170);
  vertex(200, 190);
  vertex(300, 150);
  vertex(width, 180);
  vertex(width, 280);
  vertex(0, 280);
  endShape(CLOSE);
}
```

## Что такое `beginShape()` и `vertex()`?
*   `beginShape()` — начинаем рисовать сложную форму
*   `vertex(x, y)` — добавляем точку (вершину) формы
*   `endShape(CLOSE)` — заканчиваем и соединяем последнюю точку с первой

Получается холмистая линия горизонта!

2.  Добавь ещё одно облако: скопируй строку с `ellipse` и измени координаты

### Почему функции рисуются в таком порядке?
Компьютер рисует слоями: что нарисовано раньше, то будет дальше (на заднем плане). Порядок: небо → солнце → холмы → земля → деревья → листья.
