# Шаг 4: Рисуем воду (градиент) 💧

Сделаем настоящий океан: сверху светлый, снизу тёмный. Это называется **градиент**.

## Код
```javascript
function draw() {
  for (let y = 0; y < height; y++) {
    let blue = map(y, 0, height, 255, 50);
    stroke(0, 0, blue);
    line(0, y, width, y);
  }
}
```

## Объяснение
`function draw()` — запускается 60 раз в секунду.

`for (let y = 0; y < height; y++)` — проходит по каждому пикселю сверху вниз.

`map(y, 0, height, 255, 50)` — превращает y в значение синего цвета (255 сверху, 50 снизу).

`stroke(0, 0, blue)` — устанавливает синий цвет линии.

`line(0, y, width, y)` — рисует горизонтальную линию.
