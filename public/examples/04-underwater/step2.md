# Шаг 2: Настраиваем сцену (холст) 🎨

Каждая картина начинается с холста. Наш холст — это место, где будет происходить всё действие.

## Код
```javascript
function setup() {
  createCanvas(400, 400);
}
```

## Объяснение
`function setup()` — специальная функция. Она запускается **один раз** в самом начале программы.

`createCanvas(400, 400)` — создаёт рабочую область шириной 400 и высотой 400 пикселей.

> 💡 Пиксели — это маленькие цветные точки, из которых состоит экран.
