
# Шаг 3: Запускаем рыбок в воду! 🐟

Теперь создадим первых рыбок. Сделаем их разноцветными и разного размера.

## Код
```javascript
for (let i = 0; i < 5; i++) {
  fish.push({
    x: random(width),
    y: random(height),
    size: random(20, 40),
    speed: random(1, 3),
    color: color(random(100, 255), random(100, 255), random(100, 255))
  });
}
```

## Объяснение
`for (let i = 0; i < 5; i++)` — цикл, который повторяется 5 раз.

`fish.push({...})` — добавляет новую рыбку в массив `fish`.

`x` и `y` — координаты рыбки (случайные).

`size` — размер рыбки от 20 до 40 пикселей.

`speed` — скорость движения от 1 до 3.

`color` — случайный цвет.
