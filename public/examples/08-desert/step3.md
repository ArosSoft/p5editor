# Урок 3: Барханы и песок — рисуем волны

Пустыня не была бы пустыней без песчаных барханов! Давай научимся рисовать волнистые линии и создавать глубину с помощью нескольких слоёв.

## Песок на переднем плане

Сначала нарисуем простой прямоугольник — это песок, на котором стоит наш кактус.

```javascript
// Основной песок
fill(235, 200, 170);
noStroke();
rect(0, 290, width, 110);
```

### Что означают команды?

*   **`fill(235, 200, 170)`** — устанавливает цвет заливки (песочно-бежевый).
*   **`noStroke()`** — убирает обводку у фигур.
*   **`rect(0, 290, width, 110)`** — рисует прямоугольник от x=0, y=290 шириной во весь экран (`width`) и высотой 110 пикселей.

## Дальние барханы: слои для глубины

Чтобы картина выглядела объёмной, мы рисуем несколько слоёв барханов. Дальние барханы — более бледные и маленькие. Ближние — ярче и крупнее.

```javascript
function drawDistantDunes() {
  // Несколько слоёв барханов для глубины (от дальних к ближним)
  let layers = [
    {
      baseY: 220,
      height: 20,
      color: [180, 160, 140],
      alpha: 80,
      noiseScale: 0.008
    },
    {
      baseY: 235,
      height: 28,
      color: [190, 165, 140],
      alpha: 110,
      noiseScale: 0.012
    },
    {
      baseY: 250,
      height: 35,
      color: [200, 170, 140],
      alpha: 150,
      noiseScale: 0.018
    },
    {
      baseY: 265,
      height: 42,
      color: [210, 175, 140],
      alpha: 190,
      noiseScale: 0.025
    }
  ];

  for (let layer of layers) {
    noStroke();
    fill(layer.color[0], layer.color[1], layer.color[2], layer.alpha);

    beginShape();
    vertex(0, height);
    vertex(0, layer.baseY);

    for (let x = 0; x <= width; x += 8) {
      // Шум Перлина для плавных волн дюн
      let n = noise(x * layer.noiseScale, layer.baseY * 0.01);
      let y = layer.baseY + layer.height * (n - 0.5) * 2;
      vertex(x, y);
    }

    vertex(width, layer.baseY);
    vertex(width, height);
    endShape(CLOSE);
  }
}
```

## Что такое массив объектов?

**`layers`** — это массив, в котором хранятся объекты. Каждый объект описывает один слой барханов.

### Параметры слоя:

*   **`baseY`** — базовая высота (где начинается слой по вертикали).
*   **`height`** — высота волн дюн.
*   **`color`** — цвет слоя (три числа: красный, зелёный, синий).
*   **`alpha`** — прозрачность (от 0 до 255). Чем меньше, тем прозрачнее.
*   **`noiseScale`** — масштаб шума (об этом подробнее в уроке 6).

## Рисуем сложную форму: `beginShape()` и `endShape()`

Чтобы нарисовать волнистый бархан, мы используем специальные команды:

1.  **`beginShape()`** — начинаем рисовать сложную форму.
2.  **`vertex(x, y)`** — добавляем точку (вершину) формы.
3.  **`endShape(CLOSE)`** — заканчиваем рисование и замыкаем форму.

### Как строится бархан?

```javascript
beginShape();
vertex(0, height);        // Нижний левый угол
vertex(0, layer.baseY);   // Верхний левый угол

// Перебираем точки по горизонтали
for (let x = 0; x <= width; x += 8) {
  let n = noise(x * layer.noiseScale, layer.baseY * 0.01);
  let y = layer.baseY + layer.height * (n - 0.5) * 2;
  vertex(x, y);  // Добавляем точку с волнистым Y
}

vertex(width, layer.baseY);  // Верхний правый угол
vertex(width, height);       // Нижний правый угол
endShape(CLOSE);
```

**Важно:** Мы рисуем точки каждые 8 пикселей (`x += 8`). Это делает линию плавной, но не перегружает компьютер.

## Волны с помощью шума Перлина

Строка **`let n = noise(...)`** — это магия! Функция `noise()` возвращает плавное случайное число. Мы используем его, чтобы создать волнистую линию дюн.

Формула **`y = layer.baseY + layer.height * (n - 0.5) * 2`** преобразует шум в высоту волны.
