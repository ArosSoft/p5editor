# Урок 4: Кактусы — ствол и ветви

Наша пустыня готова, но в ней нет главного героя — кактуса! Давай научимся рисовать сложные фигуры из простых частей и использовать циклы для создания ветвей.

## Рисуем кактус: основная функция

У нас есть массив `cacti`, в котором хранятся все кактусы. Чтобы нарисовать каждый из них, мы используем цикл `for...of`:

```javascript
// Рисуем кактусы
for (let cactus of cacti) {
  drawCactus(cactus);
}
```

### Что такое цикл `for...of`?

Это специальный цикл, который перебирает все элементы в массиве. Он читается почти как обычный язык:
> "Для каждого кактуса в массиве cacti — нарисуй этот кактус"

## Функция `drawCactus()` — устанавливаем позицию

```javascript
function drawCactus(cactus) {
  push();
  translate(cactus.x, cactus.y);
  drawSaguaro(cactus);
  pop();
}
```

### Команды трансформации:

*   **`push()`** — сохраняет текущее состояние холста (как закладка в книге).
*   **`translate(x, y)`** — сдвигает систему координат в точку (x, y). Теперь (0, 0) находится там, где стоит кактус.
*   **`pop()`** — восстанавливает состояние холста (возвращаемся назад).

**Зачем это нужно?** Чтобы рисовать кактус в нужном месте и не сдвинуть случайно всё остальное!

## Рисуем сагуаро: ствол и верхушка

```javascript
function drawSaguaro(cactus) {
  // Основной ствол
  fill(cactus.color);
  noStroke();
  rect(-cactus.width / 2, -cactus.height, cactus.width, cactus.height, 5);

  // Верхушка (закруглённая)
  ellipse(0, -cactus.height, cactus.width * 0.9, cactus.width * 0.6);
```

### Почему координаты отрицательные?

Помни: мы сделали `translate(cactus.x, cactus.y)`. Теперь (0, 0) — это основание кактуса. Чтобы нарисовать ствол выше, мы используем отрицательные значения Y (вверх от основания).

*   **`rect(-cactus.width / 2, -cactus.height, ...)`** — рисуем прямоугольник так, чтобы его центр был по оси X, а верх доходил до `-cactus.height`.
*   **Пятый параметр `5`** — это скругление углов прямоугольника.

## Ветви: цикл с условием

Кактус сагуаро известен своими ветвями-руками. Давай нарисуем их с помощью цикла!

```javascript
  // Руки-ветви
  for (let i = 0; i < cactus.arms; i++) {
    let armY = -cactus.height + 25 + i * 30;
    let armLen = 20 + i * 5;

    if (i % 2 === 0) {
      // Левая рука
      rect(-cactus.width / 2 - armLen, armY - 6, armLen + 8, 12, 4);
      ellipse(-cactus.width / 2 - armLen, armY, 12, 12);
    } else {
      // Правая рука
      rect(cactus.width / 2 - 8, armY - 6, armLen + 8, 12, 4);
      ellipse(cactus.width / 2 + armLen, armY, 12, 12);
    }
  }
}
```

### Разбираем алгоритм:

1.  **`for (let i = 0; i < cacti.arms; i++)`** — цикл, который выполняется столько раз, сколько ветвей у кактуса.
2.  **`let armY = -cactus.height + 25 + i * 30`** — вычисляем Y для ветви. Первая ветвь на высоте 25 пикселей от верха, каждая следующая на 30 пикселей ниже.
3.  **`let armLen = 20 + i * 5`** — длина ветви растёт с каждой итерацией (чем выше ветвь, тем она длиннее).
4.  **`if (i % 2 === 0)`** — проверяем, чётное ли число `i`. Оператор `%` возвращает остаток от деления. Если `i` делится на 2 без остатка — рисуем слева. Иначе — справа.

### Как рисуются ветви?

*   **Левая ветвь:** `rect(-cactus.width / 2 - armLen, ...)` — рисуем прямоугольник слева от ствола.
*   **Правая ветвь:** `rect(cactus.width / 2 - 8, ...)` — рисуем прямоугольник справа от ствола.
*   **`ellipse(...)`** — добавляем закругление на конце каждой ветви.
