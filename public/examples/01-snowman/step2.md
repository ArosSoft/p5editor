# Урок 2: Строим тело снеговика

Сегодня мы начнем собирать нашего снеговика. Вспомни, как лепят снеговика во дворе: сначала большой ком, потом средний, потом маленький. В коде всё точно так же.

## Команда `ellipse`
Команда `ellipse(x, y, ширина, высота)` рисует овал или круг. Если ширина и высота равны — получится круг.

## Собираем снеговика
Чтобы нарисовать три круга друг на друге, нам нужно менять координату `y`. Чем меньше число `y`, тем выше находится фигура.

Добавь этот код в функцию `draw()` после команды `background`:

```javascript
  // Рисуем тело снеговика (три белых круга)
  fill(255); // Белый цвет
  noStroke();

  // Нижний (самый большой) круг
  ellipse(200, 280, 120, 120);
  // Средний круг
  ellipse(200, 190, 90, 90);
  // Голова (верхний, маленький круг)
  ellipse(200, 115, 70, 70);
```

![Рисунок 2](/images/12.png)

### Как это работает?
Представь себе ось Y, которая идет сверху вниз.
*   `y = 115` — это верх (голова).
*   `y = 280` — это низ (ноги).
Круги рисуются от центра, поэтому они аккуратно накладываются друг на друга.
