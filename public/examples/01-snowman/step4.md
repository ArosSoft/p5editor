# Урок 4: Шляпа и руки-палки

Теперь оденем нашего снеговика. Шляпу нарисуем с помощью прямоугольников, а руки сделаем из линий.

## Шляпа (прямоугольники)
Прямоугольник рисуется командой `rect(x, y, ширина, высота)`. Важно: `x` и `y` — это координаты **верхнего левого угла** прямоугольника!

```javascript
  // Шляпа (черная)
  fill(0);
  // Поля шляпы
  rect(170, 70, 60, 10);
  // Цилиндр (верхняя часть)
  rect(180, 40, 40, 30);
```

## Руки-палки (линии)
Линия рисуется от точки до точки: `line(x1, y1, x2, y2)`.
Сделаем руки коричневыми и толстыми.

```javascript
  // Руки
  stroke(101, 67, 33); // Коричневый цвет
  strokeWeight(4);     // Толстая линия
  line(160, 170, 100, 140); // Левая рука (от туловища влево-вверх)
  line(240, 170, 300, 140); // Правая рука
```

### Почему руки рисуются в последнюю очередь?
Компьютер рисует фигуры по порядку: что написали раньше, то и будет на заднем плане. То, что позже — перекроет предыдущее. Попробуй поставить код рук **до** кода тела. Что случится? Руки окажутся под снеговиком!
