# Рисуем снеговика в p5.js 🎨

## Введение в программирование графики

В этом уроке мы познакомимся с основами рисования в библиотеке p5.js. Мы создадим простого снеговика и разберем ключевые функции для работы с графикой.

## Базовые функции p5.js

### Функция `setup()`
Эта функция выполняется **один раз** при запуске программы. Здесь мы задаем начальные настройки:

```javascript
function setup() {
    createCanvas(400, 400); // Создаем холст 400x400 пикселей
}
```

### Функция `draw()`
Выполняется **постоянно** (по умолчанию 60 раз в секунду). Весь код рисования помещается именно сюда:

```javascript
function draw() {
    background(135, 206, 235); // Заливаем фон голубым цветом
    
    fill(255); // Устанавливаем белый цвет для заливки
    noStroke(); // Отключаем обводку
    
    // Здесь будет код для рисования снеговика
}
```

## Основные команды рисования

| Функция | Описание | Пример |
|---------|----------|--------|
| `background(r, g, b)` | Заливка фона цветом | `background(135, 206, 235)` |
| `fill(r, g, b)` | Установка цвета заливки | `fill(255)` |
| `stroke(r, g, b)` | Установка цвета обводки | `stroke(0)` |
| `noStroke()` | Отключение обводки | `noStroke()` |
| `ellipse(x, y, w, h)` | Рисование эллипса | `ellipse(200, 100, 50, 50)` |

> 💡 **Цветовая модель RGB**: значения красного, зеленого и синего могут быть от 0 до 255. Например:
> - (255, 255, 255) — белый
> - (0, 0, 0) — черный
> - (135, 206, 235) — небесно-голубой
