# Урок 1: Знакомство с P5.js и первая графика

Добро пожаловать в мир креативного программирования! Сегодня мы научимся управлять компьютером с помощью кода, чтобы он рисовал за нас. Мы будем использовать библиотеку P5.js.

## Что такое P5.js?
Это инструмент для художников и дизайнеров, который позволяет создавать рисунки и анимации с помощью текстовых команд (кода).

## Наша первая программа
В окне редактора ты видишь начальный шаблон кода.

Вот две самые главные функции:
1.  **`function setup()`** — выполняется один раз в самом начале. Здесь мы настраиваем холст.
2.  **`function draw()`** — выполняется снова и снова (как кадры в мультфильме). Именно здесь мы рисуем.

### Пишем код
Давай нарисуем небо и простую фигуру. Замени шаблон на этот код в редакторе:
(Пиши без ошибок. Для работы программы не обязательно набирать текст после // - это просто комментарии для чего нужна строка или блок кода)

```javascript
function setup() {
  createCanvas(400, 400); // Создаем холст шириной 400 и высотой 400 пикселей
}

function draw() {
  background(135, 206, 235); // Заливаем фон голубым цветом (как небо)

  // Рисуем круг
  fill(255); // Белый цвет заливки
  noStroke(); // Убираем обводку
  circle(200, 200, 100); // Круг в центре холста
}
```
