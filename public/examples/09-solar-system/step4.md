# Урок 4: Методы update() и show() — движение и рисование

В этом уроке мы добавим планетам **способности**: двигаться и рисоваться на экране. Эти способности называются **методами**.

## Что такое методы?

**Метод** — это функция, которая принадлежит объекту. Другими словами, это то, что объект **умеет делать**.

**Примеры из жизни:**
- Машина умеет: `ехать()`, `сигналить()`, `тормозить()`
- Собака умеет: `лаять()`, `бежать()`, `спать()`

Наши планеты умеют:
- `update()` — обновлять положение (двигаться)
- `show()` — показываться на экране (рисоваться)

## Метод update() — движение

```javascript
update() {
  this.angle += this.speed;
}
```

### Как это работает?

| Строка | Что делает |
|--------|------------|
| `this.angle` | Берёт текущий угол планеты |
| `+=` | Прибавляет что-то к нему |
| `this.speed` | Скорость этой планеты |

**Простыми словами:** «Возьми мой текущий угол и прибавь к нему мою скорость».

### Почему планета движется?

Вспомни урок 2 про полярные координаты:
- Угол определяет, где на окружности находится планета
- Если угол растёт — планета «бежит» по кругу

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Планета с углом, который увеличивается]**

### Пример:

Было:
```javascript
this.angle = 0;      // Планета справа
this.speed = 0.01;   // Скорость
```

После одного вызова `update()`:
```javascript
this.angle = 0.01;   // Чуть сдвинулась
```

После 100 вызовов:
```javascript
this.angle = 1.0;    // Прошла заметную часть круга
```

## Метод show() — рисование

Этот метод сложнее. Он делает три вещи:

```javascript
show() {
  // 1. Вычисляем координаты X и Y
  let x = cos(this.angle) * this.distance;
  let y = sin(this.angle) * this.distance;

  // 2. Рисуем кольца (если есть)
  if (this.hasRings) {
    noFill();
    stroke(200, 180, 150, 150);
    strokeWeight(3);
    ellipse(x, y, this.radius * 2.5, this.radius * 2.5);
  }

  // 3. Рисуем саму планету
  fill(this.color);
  noStroke();
  ellipse(x, y, this.radius, this.radius);

  // 4. Рисуем название
  fill(255);
  textSize(10);
  textAlign(CENTER);
  text(this.name, x, y - this.radius - 5);
}
```

## Шаг 1: Вычисляем координаты

```javascript
let x = cos(this.angle) * this.distance;
let y = sin(this.angle) * this.distance;
```

Это формула из урока 2! Мы переводим полярные координаты (угол и расстояние) в обычные X и Y.

## Шаг 2: Рисуем кольца (если есть)

```javascript
if (this.hasRings) {
  // Рисуем кольца
}
```

### Что такое `if`?

**`if`** — это команда «если». Она выполняет код только если условие **правда** (`true`).

**Пример:**
```javascript
if (this.hasRings) {     // Если у планеты есть кольца
  drawRings();           // Нарисовать кольца
}
```

### Почему кольца рисуются ДО планеты?

Если нарисовать планету первой, она закроет кольца. Поэтому сначала кольца (на заднем плане), потом планета (на переднем).

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Порядок рисования: кольца → планета]**

### Параметры колец:

```javascript
noFill();                        // Без заливки (прозрачные внутри)
stroke(200, 180, 150, 150);      // Бежевый цвет, 150 = полупрозрачный
strokeWeight(3);                 // Толщина 3 пикселя
ellipse(x, y, ...);              // Рисуем кольцо вокруг планеты
```

## Шаг 3: Рисуем планету

```javascript
fill(this.color);        // Цвет планеты
noStroke();              // Без обводки
ellipse(x, y, this.radius, this.radius);  // Круг в координатах (x, y)
```

Всё просто: заливаем цветом и рисуем круг!

## Шаг 4: Рисуем название

```javascript
fill(255);                    // Белый цвет
textSize(10);                 // Размер текста 10 пикселей
textAlign(CENTER);            // Выравнивание по центру
text(this.name, x, y - this.radius - 5);  // Текст над планетой
```

### Почему `y - this.radius - 5`?

- `y` — центр планеты
- `- this.radius` — поднимаемся на радиус вверх (до края планеты)
- `- 5` — ещё 5 пикселей вверх (чтобы текст не прилипал)

## Как вызывать методы?

```javascript
let earth = new Planet(...);

earth.update();  // Планета двигается
earth.show();    // Планета рисуется
```
