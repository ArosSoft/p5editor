# Урок 1: Глобальные переменные и массивы

Добро пожаловать в космос! Сегодня мы создадим свою собственную Солнечную систему. Но прежде чем запускать ракеты, давай разберёмся с одним важным инструментом программиста — **массивами**.

## Что такое переменная?

**Переменная** — это как коробка с подписью. В неё можно положить что-то и потом достать.

```javascript
let radius = 10;  // Коробка с названием "radius", внутри число 10
```

- `let` — команда «дай мне коробку»
- `radius` — название коробки (переменной)
- `10` — то, что внутри

## Что такое массив?

**Массив** — это особая коробка, в которой можно хранить **много предметов сразу**. Представь себе рюкзак с отделениями для разных вещей.

```javascript
let planets = [];  // Пустой массив (пустой рюкзак)
```

- `[]` — пустой массив (пустой рюкзак)
- `planets` — название массива

## Как положить что-то в массив?

Для этого есть специальная команда `.push()`:

```javascript
planets.push("Меркурий");  // Положили в массив текст "Меркурий"
planets.push("Венера");    // Положили ещё одну планету
planets.push("Земля");     // И ещё одну
```

Теперь в массиве `planets` хранится три элемента!

## Как достать элементы из массива?

У каждого элемента в массиве есть **номер** (начинается с нуля!):

| Номер | Элемент |
|-------|---------|
| 0 | "Меркурий" |
| 1 | "Венера" |
| 2 | "Земля" |

Чтобы достать элемент, пишут так:
```javascript
let firstPlanet = planets[0];  // Достали "Меркурий"
```

> **Важно:** Счёт начинается с **нуля**, а не с единицы! Это правило во всех языках программирования.

## Глобальная переменная

В нашем коде массив создан **снаружи функций**:

```javascript
let planets = [];  // Глобальная переменная

function setup() {
  // Здесь можно использовать planets
}

function draw() {
  // И здесь тоже!
}
```

Такая переменная называется **глобальной** — она видна отовсюду. Как лампа на потолке: светит всем в комнате, а не только одному человеку.

## Зачем нам массив планет?

Мы будем хранить в массиве **объекты планет**. Каждая планета — это объект со своими свойствами:
- Цвет
- Размер
- Расстояние от Солнца
- Скорость движения

Массив позволит нам легко управлять всеми планетами сразу!
