# Урок 3: Класс Planet — чертёж для создания планет

В этом уроке мы узнаем, как создавать **объекты** с помощью **классов**. Это одна из самых мощных идей в программировании!

## Что такое класс?

**Класс** — это как **чертёж** или **шаблон**. По одному чертежу можно сделать много одинаковых предметов.

**Пример из жизни:**
- **Чертёж** стула — это класс
- **Конкретные стулья**, сделанные по этому чертежу — это объекты

## Объявление класса

Класс объявляется ключевым словом `class`:

```javascript
class Planet {
  // Внутри класса пишется код
}
```

## Конструктор — особая функция

Внутри класса есть специальная функция — **конструктор**. Она запускается, когда создаётся новый объект.

```javascript
class Planet {
  constructor(c, r, d, s, hasRings, name) {
    this.color = c;           // Цвет планеты
    this.radius = r;          // Радиус в пикселях
    this.distance = d;        // Расстояние от Солнца
    this.speed = s;           // Скорость вращения
    this.angle = random(TWO_PI); // Случайный угол старта
    this.hasRings = hasRings; // Есть ли кольца
    this.name = name;         // Название планеты
  }
}
```

### Что такое `this`?

**`this`** — это слово означает «мой собственный». Как будто планета говорит:
- «**Мой** цвет»
- «**Мой** размер»
- «**Моё** расстояние от Солнца»

Каждая планета будет иметь **свои собственные** значения этих свойств.

## Параметры конструктора

Разберём, что означает каждая буква:

| Параметр | Полное название | Что означает | Пример |
|----------|----------------|--------------|--------|
| `c` | color | Цвет планеты | `color(255, 100, 100)` — красный |
| `r` | radius | Радиус (размер) | `10` — маленькая, `22` — большая |
| `d` | distance | Расстояние от Солнца | `80` — близко, `440` — далеко |
| `s` | speed | Скорость вращения | `0.02` — быстро, `0.002` — медленно |
| `hasRings` | has rings | Есть ли кольца | `true` — да, `false` — нет |
| `name` | name | Название | `"Земля"`, `"Марс"` |

## Случайный угол старта

```javascript
this.angle = random(TWO_PI);
```

**Зачем это нужно?**
- `TWO_PI` — это полный круг (360°) в радианах
- `random(TWO_PI)` — случайное число от 0 до полного круга

Благодаря этому планеты появляются в **разных местах** орбиты, а не все в одной точке!

## Как создать объект планеты?

Используется ключевое слово `new`:

```javascript
let earth = new Planet(
  color(100, 150, 255),  // Голубой цвет
  16,                     // Размер
  160,                    // Расстояние от Солнца
  0.01,                   // Скорость
  false,                  // Нет колец
  "Земля"                 // Название
);
```

Теперь у нас есть объект `earth` — это конкретная планета, созданная по чертежу `Planet`.

## Свойства объекта

У созданной планеты можно читать и менять свойства:

```javascript
console.log(earth.radius);     // 16 — читаем размер
console.log(earth.name);       // "Земля" — читаем название
earth.speed = 0.02;            // Меняем скорость на большую
```

## Почему это удобно?

Без классов пришлось бы создавать много отдельных переменных:

```javascript
// ❌ Плохо: много отдельных переменных
let planet1Color, planet1Radius, planet1Distance...
let planet2Color, planet2Radius, planet2Distance...
```

С классом — всё аккуратно:

```javascript
// ✅ Хорошо: каждый объект хранит свои данные
let planet1 = new Planet(...);
let planet2 = new Planet(...);
```
