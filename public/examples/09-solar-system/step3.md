# Часть 3. Функция draw() — постоянное обновление

## Что такое draw()?
`draw()` работает **бесконечно**, как мультик (~60 раз в секунду).

> **Аналогия:** Блокнот с картинками. Быстро перелистываешь — получается движение.

```javascript
function draw() {
  background(0);
  push();
  translate(width / 2, height / 2);
  
  // Солнце и орбиты...
  
  for (let planet of planets) {
    planet.update();
    planet.show();
  }
  pop();
}
```

## По шагам:

| Шаг | Код | Зачем? |
|-----|-----|--------|
| 1 | `background(0)` | Очистить экран (чёрный) |
| 2 | `translate(...)` | Переместить центр в середину |
| 3 | `push()` / `pop()` | Сохранить и вернуть положение |
| 4 | `planet.update()` | Сдвинуть планету по орбите |
| 5 | `planet.show()` | Нарисовать планету |
