# Урок 2: Полярная система координат — секрет движения планет

В этом уроке мы узнаем, как математика помогает нам двигать планеты по кругу! Это **самый важный урок** для понимания того, как работает наша Солнечная система.

## Две системы координат

### 1. Декартова система (прямоугольная)

Ты уже знаком с ней! Это когда положение точки задаётся двумя числами: `x` и `y`.

```javascript
ellipse(100, 200, 50, 50);  // x=100, y=200
```

**Проблема:** Попробуй заставить этот круг двигаться **по кругу** вокруг центра. Нужно постоянно менять и `x`, и `y` одновременно. Это сложно!

### 2. Полярная система (круговая)

А что если вместо `x` и `y` использовать другие координаты:
- **Расстояние** от центра (назовём его `r` — от слова «radius», радиус)
- **Угол** поворота (назовём его `angle` или греческой буквой `θ` — тета)

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Сравнение декартовой и полярной систем координат]**

## Как это работает?

Представь, что ты держишь верёвку с шариком и крутишь её вокруг себя:
- **Длина верёвки** — это расстояние `r` (радиус)
- **Угол поворота** — это `angle` (насколько ты повернул руку)

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Человек крутит шарик на верёвке]**

## Волшебные формулы: sin и cos

Чтобы перевести полярные координаты (расстояние и угол) в обычные (x и y), нужны две математические функции:

```javascript
let x = cos(angle) * distance;  // Косинус угла × расстояние
let y = sin(angle) * distance;  // Синус угла × расстояние
```

### Что такое sin и cos?

Не пугайся! Это просто «табличные числа», которые компьютер знает наизусть.

**Правило для запоминания:**
- `cos(angle)` — говорит, насколько точка смещена **вправо-влево**
- `sin(angle)` — говорит, насколько точка смещена **вверх-вниз**

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Единичная окружность с подписями sin и cos]**

### Пример:

Если угол = 0°, планета справа:
```javascript
angle = 0;
x = cos(0) * 100;  // 1 × 100 = 100 (справа)
y = sin(0) * 100;  // 0 × 100 = 0 (по центру)
```

Если угол = 90° (π/2 радиан), планета внизу:
```javascript
angle = PI / 2;
x = cos(PI/2) * 100;  // 0 × 100 = 0 (по центру)
y = sin(PI/2) * 100;  // 1 × 100 = 100 (внизу)
```

## Что такое радианы?

В программировании углы измеряются не в градусах, а в **радианах**.

| Градусы | Радианы |
|---------|---------|
| 0° | 0 |
| 90° | π/2 (PI / 2) |
| 180° | π (PI) |
| 270° | 3π/2 |
| 360° | 2π (TWO_PI) |

**Почему?** Так удобнее для математики. Полный круг = `TWO_PI` радиан.

## Движение по орбите

Теперь самое интересное! Чтобы планета **двигалась**, нужно понемногу увеличивать угол:

```javascript
angle = angle + speed;  // speed — скорость вращения
```

Каждый кадр мы добавляем к углу немного. И планета движется по кругу!

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Планета движется по орбите с увеличением угла]**

## Полный код движения планеты

Вот как выглядит весь алгоритм:

```javascript
// 1. Увеличиваем угол
this.angle = this.angle + this.speed;

// 2. Переводим полярные координаты в x и y
let x = cos(this.angle) * this.distance;
let y = sin(this.angle) * this.distance;

// 3. Рисуем планету в новых координатах
ellipse(x, y, this.radius, this.radius);
```

## Почему это работает?

| Шаг | Что происходит |
|-----|----------------|
| `angle += speed` | Угол растёт → планета «бежит» по кругу |
| `cos(angle) * distance` | Вычисляем X координату на окружности |
| `sin(angle) * distance` | Вычисляем Y координату на окружности |
| `ellipse(x, y, ...)` | Рисуем планету в новой точке |
