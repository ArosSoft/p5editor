# Урок 5: Функция setup() — создаём Солнечную систему

В этом уроке мы наполним нашу Солнечную систему планетами! Для этого используем специальную функцию `setup()`.

## Что такое setup()?

**`setup()`** — это функция, которая выполняется **только один раз** при запуске программы.

**Аналогия из жизни:**
- `setup()` — это как подготовка к уроку: ты один раз достаёшь учебник, тетрадь и ручки
- Потом начинается урок (`draw()`), который продолжается много раз

## Структура setup()

```javascript
function setup() {
  createCanvas(600, 600);  // 1. Создаём холст

  // 2. Добавляем планеты в массив
  planets.push(new Planet(color(200, 150, 100), 10, 80, 0.02, false, "Меркурий"));
  planets.push(new Planet(color(255, 200, 100), 14, 120, 0.015, false, "Венера"));
  planets.push(new Planet(color(100, 150, 255), 16, 160, 0.01, false, "Земля"));
  planets.push(new Planet(color(255, 100, 100), 12, 200, 0.008, false, "Марс"));
  planets.push(new Planet(color(200, 150, 100), 22, 260, 0.005, false, "Юпитер"));
  planets.push(new Planet(color(210, 180, 140), 20, 320, 0.004, true, "Сатурн"));
  planets.push(new Planet(color(150, 200, 200), 18, 380, 0.003, false, "Уран"));
  planets.push(new Planet(color(100, 100, 255), 17, 440, 0.002, false, "Нептун"));
}
```

## Шаг 1: Создаём холст

```javascript
createCanvas(600, 600);
```

Эта команда создаёт «лист бумаги» размером 600×600 пикселей.

## Шаг 2: Добавляем планеты

Каждая строка создаёт новую планету:

```javascript
planets.push(new Planet(...));
```

Разберём по частям:
1. `new Planet(...)` — создаёт новую планету
2. `planets.push(...)` — кладёт её в массив

## Интересные закономерности

### 1. Чем дальше, тем медленнее

Посмотри на скорость планет:
- Меркурий (близко): `0.02` — самый быстрый
- Нептун (далеко): `0.002` — самый медленный

**Почему?** В реальной жизни планеты, которые ближе к Солнцу, вращаются быстрее. Это закон физики!

### 2. Чем дальше, тем больше расстояние

Расстояние увеличивается с каждым шагом:
- Меркурий: `80`
- Венера: `120` (+40)
- Земля: `160` (+40)
- ...
- Нептун: `440`

Это нужно, чтобы орбиты не накладывались друг на друга.

### 3. Размеры планет

- Юпитер и Сатурн — самые большие (22 и 20)
- Меркурий и Марс — маленькие (10 и 12)
- Остальные — средние (14–18)

Это соответствует реальным размерам планет!

## Цвета планет

Цвет задаётся тремя числами: `color(R, G, B)`

| Цвет | R (красный) | G (зелёный) | B (синий) | Результат |
|------|-------------|-------------|-----------|-----------|
| Меркурий | 200 | 150 | 100 | Коричневый |
| Венера | 255 | 200 | 100 | Оранжевый |
| Земля | 100 | 150 | 255 | Голубой |
| Марс | 255 | 100 | 100 | Красный |
| Сатурн | 210 | 180 | 140 | Бежевый |
| Уран | 150 | 200 | 200 | Бирюзовый |
| Нептун | 100 | 100 | 255 | Синий |

**Правило:**
- Больше красного → теплее цвет
- Больше синего → холоднее цвет
- Все три равны → серый оттенок

## Почему Сатурн с кольцами?

Только у Сатурна последний параметр `true`:

```javascript
planets.push(new Planet(..., true, "Сатурн"));
```

В методе `show()` есть проверка:
```javascript
if (this.hasRings) {
  // Рисуем кольца
}
```

Поэтому кольца появятся только у Сатурна!
