# Часть 6. Развитие проекта

## Добавь остальные планеты

В Солнечной системе **8 планет**, а у нас только 5. Добавь в `setup()`:

```javascript
// Сатурн
planets.push(new Planet(color(210, 180, 140), 20, 320, 0.004));

// Уран
planets.push(new Planet(color(150, 200, 200), 18, 380, 0.003));

// Нептун
planets.push(new Planet(color(100, 100, 255), 17, 440, 0.002));
```

| Планета | Цвет | Размер | Расстояние | Скорость |
|---------|------|--------|------------|----------|
| Сатурн | 210, 180, 140 | 20 | 320 | 0.004 |
| Уран | 150, 200, 200 | 18 | 380 | 0.003 |
| Нептун | 100, 100, 255 | 17 | 440 | 0.002 |

> **Чем дальше планета, тем медленнее вращается!**

## 💍 Кольца Сатурна

**1.** Добавь параметр `hasRings` в конструктор:
```javascript
constructor(c, r, d, s, hasRings) {
  // ...
  this.hasRings = hasRings;
}
```

**2.** При создании Сатурна передай `true`:
```javascript
planets.push(new Planet(..., true));  // Сатурн с кольцами
```

**3.** В методе `show()` рисуй кольца:
```javascript
if (this.hasRings) {
  noFill();
  stroke(200, 180, 150, 150);
  strokeWeight(3);
  ellipse(x, y, this.radius * 2.5, this.radius * 2.5);
}
```

> **Важно:** Кольца рисуй **ДО** планеты!

## 🏷️ Подписи планет

Добавь параметр `name` и в `show()`:
```javascript
fill(255);
textSize(10);
textAlign(CENTER);
text(this.name, x, y - this.radius - 5);
```
