# Урок 6: Функция draw() — оживляем Вселенную

В этом уроке мы узнаем, как заставить планеты двигаться! За анимацию отвечает функция `draw()`.

## Что такое draw()?

**`draw()`** — это функция, которая выполняется **бесконечно**, около 60 раз в секунду.

**Аналогия из жизни:**
- `draw()` — это как кадры в мультфильме
- Каждый кадр — это один вызов функции
- Если быстро показывать кадры — получается движение

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Блокнот с картинками как анимация]**

## Структура draw()

```javascript
function draw() {
  // 1. Очищаем экран
  background(0);

  // 2. Сохраняем текущее положение
  push();

  // 3. Перемещаем центр координат
  translate(width / 2, height / 2);

  // 4. Рисуем Солнце и орбиты
  // ... (об этом в следующем уроке)

  // 5. Обновляем и рисуем каждую планету
  for (let planet of planets) {
    planet.update();
    planet.show();
  }

  // 6. Восстанавливаем положение
  pop();
}
```

## Шаг 1: Очистка экрана

```javascript
background(0);  // 0 = чёрный цвет
```

**Зачем?**
Каждый кадр нужно стирать предыдущий! Иначе планеты оставят след на экране.

**Эксперимент:** Попробуй убрать эту строку. Планеты будут оставлять цветные следы!

## Шаг 2-3: Перемещение центра координат

### Проблема

По умолчанию центр координат (0, 0) находится в **левом верхнем углу**:

```
(0, 0) → x →
  ↓
  y
```

Но нам нужно, чтобы центр был **в середине холста**, где Солнце!

### Решение

Используем две команды:

```javascript
push();                              // Сохранить текущее положение
translate(width / 2, height / 2);    // Сдвинуть центр в середину
```

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Сдвиг центра координат в середину]**

### Что делает translate()?

```javascript
translate(width / 2, height / 2);
```

- `width` — ширина холста (600)
- `height` — высота холста (600)
- `width / 2` — половина ширины (300)
- `height / 2` — половина высоты (300)

Команда `translate()` сдвигает **начало координат** в точку (300, 300).

Теперь:
- (0, 0) — это центр холста!
- (100, 0) — 100 пикселей вправо от центра
- (0, 100) — 100 пикселей вниз от центра

### Зачем нужны push() и pop()?

```javascript
push();     // Запомнить, где был центр
// ... меняем центр ...
pop();      // Вернуть центр как было
```

**Аналогия:**
- `push()` — положить закладку в книгу
- `pop()` — вернуться к закладке

Это нужно, чтобы не сломать координаты для других объектов!

## Шаг 5: Цикл for...of

Самая важная часть! Здесь мы заставляем планеты двигаться:

```javascript
for (let planet of planets) {
  planet.update();
  planet.show();
}
```

### Что такое цикл?

**Цикл** — это команда «повтори для каждого».

**Пример из жизни:**
- «Для каждого ученика в классе: вызови к доске»
- «Для каждой планеты в массиве: обнови и нарисуй»

### Как работает for...of?

```javascript
for (let planet of planets) {
  // Код внутри выполняется для каждой планеты
}
```

| Итерация | planet = | Что происходит |
|----------|----------|----------------|
| 1 | Меркурий | `update()`, `show()` для Меркурия |
| 2 | Венера | `update()`, `show()` для Венеры |
| 3 | Земля | `update()`, `show()` для Земли |
| ... | ... | ... |
| 8 | Нептун | `update()`, `show()` для Нептуна |

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Цикл перебирает планеты по очереди]**

### Что происходит внутри цикла?

```javascript
planet.update();   // 1. Планета двигается по орбите
planet.show();     // 2. Планета рисуется на экране
```

Сначала **все** планеты двигаются, потом **все** рисуются.

## Как работает анимация?

Вот что происходит за 1 секунду:

| Время | Что происходит |
|-------|----------------|
| 0 мс | `draw()` вызван в первый раз: планеты обновились и нарисовались |
| 16 мс | `draw()` вызван второй раз: планеты чуть сдвинулись |
| 32 мс | `draw()` вызван третий раз: планеты ещё сдвинулись |
| ... | ... |
| 1000 мс | `draw()` вызван ~60 раз! |

Получается плавное движение!

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: 60 кадров в секунду]**

## Порядок важен!

Обрати внимание на порядок в `draw()`:

```javascript
background(0);     // 1. Сначала очистить
push();            // 2. Сохранить
translate(...);    // 3. Сдвинуть центр
// ... рисовать ...
pop();             // 4. Вернуть назад
```

Если поменять порядок — всё сломается!
