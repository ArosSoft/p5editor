# Часть 1: Подготовка сцены — переменные и настройка

Перед тем как начать рисовать, нужно создать «хранилища» для данных и настроить сцену.  
Мы объявляем **массив** `raindrops`, в котором будем хранить информацию о каждой капле дождя.  
Переменная `lightning` будет отвечать за яркость вспышки молнии.

```js
let raindrops = [];
let lightning = 0;

function setup() {
  createCanvas(400, 400);
  
  // Создаём капли дождя
  for (let i = 0; i < 100; i++) {
    raindrops.push({
      x: random(width),
      y: random(height),
      speed: random(5, 10),
      length: random(10, 20)
    });
  }
}
```

Функция `setup()` выполняется один раз в начале программы.  
Здесь мы задаём размер холста и заполняем массив `raindrops` 100 каплями.  
Каждая капля — это **объект** с координатами `x`, `y`, скоростью падения `speed` и длиной `length`.
