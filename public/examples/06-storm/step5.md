# Часть 5: Рисование луж

Внизу холста рисуем прямоугольник — это лужа.  
Цвет — синий с прозрачностью `100`, чтобы было видно, что вода слегка прозрачна.  
Прямоугольник занимает всю ширину экрана и высоту 20 пикселей.

```js
// Лужи внизу
  fill(100, 100, 200, 100);
  rect(0, height - 20, width, 20);
```
