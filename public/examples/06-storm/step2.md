# Часть 2: Рисование фона и управление молнией

В функции `draw()`, которая выполняется 60 раз в секунду, мы начинаем с заливки фона тёмно-синим цветом.  
Затем случайным образом с вероятностью 0.005 «включаем» молнию, устанавливая `lightning` в 255.

Если молния активна (`lightning > 0`), рисуем белый полупрозрачный прямоугольник поверх всего холста — это создаёт эффект вспышки.  
После каждого кадра яркость уменьшается на 5, и вспышка постепенно исчезает.

```js
function draw() {
  // Тёмное небо
  background(30, 30, 50);
  
  // Молния
  if (random() < 0.005) {
    lightning = 255;
  }
  
  if (lightning > 0) {
    fill(255, 255, 255, lightning);
    rect(0, 0, width, height);
    lightning -= 5;
  }
  // Сдесь будет весь осташийся код.
}
```
