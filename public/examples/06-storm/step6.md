# Часть 6: Круги на воде

Чтобы добавить оживления, рисуем круги, которые появляются на поверхности луж.  
Используем цикл для создания 5 кругов.  
Координата `x` каждого круга вычисляется с помощью `(frameCount * 2 + i * 50) % width`.  
Благодаря этому круги плавно движутся слева направо и возвращаются в начало.

`frameCount` — это счётчик кадров, который увеличивается с каждым обновлением экрана.

```js
// Круги на лужах
  fill(255, 255, 255, 100);
  for (let i = 0; i < 5; i++) {
    let x = (frameCount * 2 + i * 50) % width;
    ellipse(x, height - 15, 5, 5);
  }
```
