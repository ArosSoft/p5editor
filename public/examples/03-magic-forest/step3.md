# Урок 3: Создаём светлячков

Теперь добавим волшебства — создадим светлячков, которые будут летать и мерцать!

## Чем светлячок отличается от дерева?
Дерево просто стоит на месте. А светлячок должен **двигаться**! Поэтому нам нужно хранить больше информации:

*   `x`, `y` — где находится светлячок
*   `speedX`, `speedY` — насколько быстро он летит в каждом кадре
*   `brightness` — насколько ярко он светится

## Создаём массив светлячков
Добавь этот код в `setup()` после цикла с деревьями:

```javascript
  // Создаём светлячков
  for (let i = 0; i < 20; i++) {
    fireflies.push({
      x: random(width),
      y: random(height),
      speedX: random(-1, 1),
      speedY: random(-1, 1),
      brightness: random(100, 255)
    });
  }
```

## Что такое отрицательная скорость?
Скорость — это число, которое показывает, насколько сильно изменится позиция.
*   `speedX: 1` — светлячок летит вправо на 1 пиксель за кадр
*   `speedX: -1` — светлячок летит влево на 1 пиксель за кадр
*   `speedX: 0` — светлячок не двигается по горизонтали

`random(-1, 1)` может вернуть и отрицательное, и положительное число. Так мы задаём случайное направление!

## Как работает движение?
Чтобы объект двигался, нужно в каждом кадре прибавлять скорость к позиции:

```javascript
x = x + speedX;  // Новая позиция X
y = y + speedY;  // Новая позиция Y
```

Это как в жизни: если ты идёшь со скоростью 5 км/ч, то через час ты будешь на 5 км дальше.

Программисты выбирают число так, чтобы было красиво и компьютер не тормозил. 20 — хорошее число!
