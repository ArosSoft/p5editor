# Урок 6: Анимация светлячков

Самая волшебная часть — заставим светлячков летать и мерцать!

## Движение светлячков
Вспомни: чтобы объект двигался, нужно прибавлять его скорость к позиции.

Добавь этот код в `draw()` после рисования деревьев:

```javascript
  // Рисуем светлячков
  for (let firefly of fireflies) {
    // Двигаем светлячка
    firefly.x += firefly.speedX;
    firefly.y += firefly.speedY;

    // Отскакиваем от краёв
    if (firefly.x < 0 || firefly.x > width) firefly.speedX *= -1;
    if (firefly.y < 0 || firefly.y > height) firefly.speedY *= -1;

    // Мерцание
    firefly.brightness = 150 + 105 * sin(frameCount * 0.1 + firefly.x);

    // Рисуем светлячка
    fill(255, 255, 100, firefly.brightness);
    noStroke();
    ellipse(firefly.x, firefly.y, 5, 5);

    // Светящийся ореол
    fill(255, 255, 100, firefly.brightness * 0.3);
    ellipse(firefly.x, firefly.y, 15, 15);
  }
```

## Как работает движение?
`firefly.x += firefly.speedX` — это то же самое, что `firefly.x = firefly.x + firefly.speedX`.
Мы берём текущую позицию и прибавляем скорость.

## Отскок от краёв
Чтобы светлячки не улетали за экран, проверяем их позицию:

```javascript
if (firefly.x < 0 || firefly.x > width) firefly.speedX *= -1;
```

Это читается как: "Если светлячок слева от 0 ИЛИ справа от ширины — разверни скорость по горизонтали".

`*= -1` — умножает скорость на -1, то есть разворачивает в противоположную сторону.

## Что такое `frameCount`?
`frameCount` — это специальная переменная, которая считает, сколько кадров нарисовал компьютер.
*   В начале: `frameCount = 0`
*   Через 1 секунду: `frameCount = 60` (если 60 кадров в секунду)
*   Через 10 секунд: `frameCount = 600`

## Мерцание с помощью `sin`
Функция `sin()` возвращает число от -1 до 1, которое плавно меняется туда-сюда, как волна.

```javascript
sin(frameCount * 0.1 + firefly.x)
```

*   `frameCount * 0.1` — медленно меняющееся число (волна)
*   `+ firefly.x` — каждый светлячок мерцает в своём ритме

Мы умножаем на 105 и прибавляем 150, чтобы получить яркость от 45 до 255.

## Прозрачность
Четвёртое число в `fill(255, 255, 100, firefly.brightness)` — это прозрачность:
*   0 — полностью прозрачный
*   255 — полностью видимый

Мы используем яркость как прозрачность, чтобы светлячок плавно появлялся и исчезал!
