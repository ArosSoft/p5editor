# Урок 5: Рисуем деревья

Теперь, когда фон готов, давайте нарисуем деревья из нашего массива!

## Цикл for...of
Чтобы пройти по всем элементам массива, удобно использовать цикл `for...of`:

```javascript
for (let tree of trees) {
  // Здесь tree — это одно дерево из массива
  // Можно обращаться к tree.x, tree.y, tree.height, tree.width
}
```

Это читается как: "Для каждого дерева в массиве trees сделай..."

## Рисуем ствол и крону
Дерево состоит из двух частей:
1.  **Ствол** — коричневый прямоугольник
2.  **Крона** — зелёные эллипсы (как облака)

Добавь этот код в `draw()` после рисования травы:

```javascript
  // Рисуем деревья
  for (let tree of trees) {
    // Ствол
    fill(101, 67, 33);  // Коричневый
    rect(tree.x - tree.width / 2, tree.y - tree.height, tree.width, tree.height);

    // Крона
    fill(34, 139, 34);  // Зелёный
    ellipse(tree.x, tree.y - tree.height - 20, tree.width * 2, tree.width * 1.5);
    ellipse(tree.x - 15, tree.y - tree.height - 5, tree.width * 1.5, tree.width * 1.2);
    ellipse(tree.x + 15, tree.y - tree.height - 5, tree.width * 1.5, tree.width * 1.2);
  }
```

## Почему `tree.x - tree.width / 2`?
Помнишь, что `rect()` рисует от верхнего левого угла? Но мы хотим, чтобы дерево было **по центру** своей координаты `x`!

Поэтому мы сдвигаем начало прямоугольника влево на половину ширины:
```
tree.x - tree.width / 2
```

Тогда центр ствола будет точно в точке `tree.x`.

## Почему три эллипса для кроны?
Один эллипс выглядит скучно. Три эллипса, расположенных рядом, создают форму, похожую на пушистую крону дерева:
*   Один большой — в центре
*   Два поменьше — слева и справа

### Порядок рисования важен!
Компьютер рисует всё по порядку. Если нарисовать деревья **до** травы, они окажутся за ней. Попробуй поменять местами код травы и деревьев — что получится?
