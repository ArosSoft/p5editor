
### Часть 1: Подготовка сцены и массив
Прежде чем рисовать, мы создаем «хранилище» для всех цветов. В программировании такое хранилище называется **массив**. Представь, что это коробка, в которую мы будем складывать инструкции по рисованию каждого цветка. Функция `setup()` запускается один раз в самом начале.

```javascript
let flowers = []; // Пустая коробка для цветов

function setup() {
  createCanvas(400, 400); // Создаем холст 400 на 400 пикселей
}
```
> **[Место для изображения: Пустой холст и иконка коробки (массива) рядом]**
