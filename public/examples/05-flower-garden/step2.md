### Часть 2: Создание (генерация) цветов
 Мы используем **цикл** `for`, чтобы не рисовать 10 цветов руками, а создать их автоматически. Каждый цветок — это **объект** (набор характеристик): у него есть координата X, Y, цвет лепестков, размер и угол наклона. `random()` делает сад разнообразным.

```javascript
for (let i = 0; i < 10; i++) {
  flowers.push({ // Добавляем новый цветок в нашу коробку (массив)
    x: random(50, 350),        // Позиция по горизонтали
    y: random(300, 350),       // Позиция по вертикали (ближе к земле)
    petalColor: color(random(255), random(255), random(255)), // Случайный цвет
    centerColor: color(255, 255, 0), // Желтая серединка
    size: random(20, 70),      // Размер от 20 до 70
    angle: 0                   // Начальный угол поворота
  });
}
```
