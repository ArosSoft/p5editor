### Часть 3: Фон и статичные объекты
 Функция `draw()` работает как бесконечный мультфильм (60 раз в секунду). Сначала мы всегда перерисовываем фон, чтобы старые рисунки исчезали. Здесь мы рисуем небо (`background`), солнце (`ellipse`) и зеленую траву (`rect`).

```javascript
function draw() {
  background(135, 206, 235); // Заливка небесно-голубым

  fill(255, 255, 0);         // Включаем желтый цвет
  ellipse(350, 50, 60, 60);  // Рисуем солнце (x, y, ширина, высота)

  fill(34, 100, 34);         // Включаем темно-зеленый
  rect(0, 200, width, 200);  // Рисуем траву (x, y, ширина, высота)
}
```
> **[Место для изображения: Схема координат холста: где находится трава (y=200) и солнце (350,50)]**
