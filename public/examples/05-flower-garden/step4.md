### Часть 4: Анимация (колыхание цветов)
 Чтобы цветы колыхались на ветру, мы меняем свойство `angle` (угол) каждого цветка. `frameCount` считает кадры. Формула `sin` создает плавное движение туда-обратно. Чем левее цветок (`flower.x`), тем немного отличается ритм.

```javascript
for (let flower of flowers) {
  // Наклоняем цветок по синусоиде. Чем больше frameCount, тем сильнее наклон.
  flower.angle = sin(frameCount * 0.02 + flower.x) * 0.1;
  
  push();        // Сохраняем текущие настройки холста
  translate(flower.x, flower.y); // Перемещаем точку рисования к цветку
  rotate(flower.angle);          // Поворачиваем цветок
  // ... здесь будет рисование стебля и лепестков ...
  pop();         // Возвращаем настройки обратно (чтобы другие цветы не съехали)
}
```
