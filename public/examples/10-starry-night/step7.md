# Шаг 7: Ковш на небе

Какое же ночное небо без созвездий? Давай нарисуем самое известное — **Большую Медведицу** (её ещё называют просто «Ковш»).

**Как мы это сделаем?**

1.  **Звёзды созвездия:** Мы заранее знаем координаты главных звёзд ковша. Мы сохранили их в виде точек в массиве `points`.
2.  **Соединяем линиями:** Сначала мы проходим по всем точкам по порядку и соединяем их линиями. Линии мы делаем полупрозрачными (`100`), чтобы они не перекрывали звёзды.
3.  **Рисуем сами звёзды:** Поверх линий, в тех же точках, мы рисуем яркие жёлтые кружочки.

![Рисунок](images/107.PNG)
*Рисунок 5. На небе появился ковш Большой Медведицы!*
```js
// Настройки для рисования созвездия
stroke(255, 255, 255, 100);  // полупрозрачный белый
strokeWeight(1);              // тонкая линия
noFill();                     // без заливки

// Координаты звёзд Большой Медведицы
let points = [
  [70, 67], [100, 65], [120, 70], [157, 80],
  [160, 108], [200, 110], [210, 79], [157, 80]
];

// Рисуем линии между звёздами
for (let i = 0; i < points.length - 1; i++) {
  line(points[i][0], points[i][1], points[i+1][0], points[i+1][1]);
}

// Рисуем сами звёзды
for (let p of points) {
  fill(255, 255, 200);  // жёлтоватый цвет
  ellipse(p[0], p[1], 5, 5);  // рисуем кружок диаметром 5
}
```
