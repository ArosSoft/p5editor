# Шаг 4: Волшебное мерцание

Холст очищен, и теперь мы снова рисуем все наши звёзды. Но на этот раз мы не просто ставим точки, а заставляем их **мерцать**!

**Как работает волшебство?**

1.  Мы проходим по списку `stars` и берём информацию о каждой звезде.
2.  Для каждой звезды мы по-новому вычисляем её яркость (`brightness`). В этом нам помогает математическая формула с функцией **sin**.
    *   Не пугайся! Проще говоря, эта формула заставляет яркость каждой звезды плавно меняться: то становиться ярче, то тусклее. Это и есть мерцание.
    *   Благодаря `twinkleSpeed` и координате `x`, все звёзды мерцают по-разному.
3.  Мы говорим компьютеру: «Нарисуй кружок (эллипс) в координатах звезды, белым цветом, с той яркостью, которую мы только что вычислили». Команда `noStroke()` убирает у кружка контур.

---

**[ВСТАВИТЬ ИЗОБРАЖЕНИЕ: Тот же холст со звёздами, но над некоторыми из них нарисованы "волны" или значки "пульсации", показывающие, что яркость меняется]**
*Рисунок 3. Звёзды начинают мерцать! Их яркость постоянно меняется.*
```js
// Рисуем все звёзды из массива stars
for (let star of stars) {
  // Вычисляем новую яркость для мерцания
  star.brightness = 150 + 105 * sin(frameCount * star.twinkleSpeed + star.x);
  
  // Устанавливаем белый цвет с текущей яркостью
  fill(255, 255, 255, star.brightness);
  
  // Рисуем звезду без контура
  noStroke();
  ellipse(star.x, star.y, star.size, star.size);
}
```
