# Шаг 3: Оживляем небо (вечный двигатель)

Если бы мы нарисовали звёзды только один раз, они бы так и висели неподвижно. А мы хотим, чтобы они мерцали, а падающие звёзды — падали!

Для этого существует функция `draw()`. Она работает как видеокамера: срабатывает снова и снова, 60 раз в секунду, создавая движущуюся картинку.

**Что мы делаем в этом шаге?**

Самая первая команда в `draw()` — `background(0);`. Она закрашивает весь холст чёрным цветом в начале каждого нового кадра. Это как стереть старый рисунок с доски, чтобы нарисовать новый. Если этого не делать, следы от движущихся объектов растянутся по всему экрану.

Представь, что ты рисуешь мультик: каждый кадр ты рисуешь заново, немного меняя положение героев.
```js
function draw() {
  // Закрашиваем фон чёрным цветом в начале каждого кадра
  background(0);
  
  // Здесь будет остальной код рисования
}
```
