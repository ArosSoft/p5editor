---

## 💡 Советы:

1. **`prompt()` возвращает строку** — всегда превращай в число через `Number()`, если нужно число
2. **`console.log()`** — твой друг! Используй для проверки значений
3. **`keyPressed()` vs `keyIsDown()`** — первая срабатывает один раз, второе работает пока нажато
