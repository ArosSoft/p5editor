# Урок 1: Создаём пустыню — переменные и массивы

Добро пожаловать в новую приключенческую программу! Сегодня мы нарисуем жаркую пустыню с кактусами. Но не простую, а с настоящим ветром, который колышет растения!

## Что нового мы узнаем?
1.  **Переменные** — это как коробочки, в которых хранятся числа или другие данные.
2.  **Массивы** — это как список дел или коробка, где можно хранить много предметов сразу.
3.  **Функции** — это мини-программы, которые выполняют одну задачу (например, "нарисовать кактус").

## Начало кода: объявляем переменные

В самом верху файла, до всех функций, мы пишем:

```javascript
let cacti = [];
let windStrength = 0;
```

### Что здесь происходит?

*   **`let cacti = []`** — мы создали пустой массив с именем `cacti`. В него мы будем складывать кактусы. Представь, что это коробка для игрушек — пока пустая, но скоро мы положим туда одного кактуса.
*   **`let windStrength = 0`** — переменная для силы ветра. Сейчас ветер равен 0 (штиль), но потом мы заставим его меняться.

## Функция `setup()` — настраиваем холст

Функция `setup()` выполняется **один раз** при запуске программы. Здесь мы создаём холст и добавляем первый кактус.

```javascript
function setup() {
  createCanvas(400, 400);

  // Создаём один кактус (большой и ближний)
  cacti.push(createCactus(150, 340));
}
```

### Разбираем по строчкам:

1.  **`createCanvas(400, 400)`** — создаёт холст шириной 400 пикселей и высотой 400 пикселей. Это как лист бумаги для рисования.
2.  **`cacti.push(...)`** — команда `push()` добавляет что-то в массив. Мы кладём в нашу "коробку" один кактус.
3.  **`createCactus(150, 340)`** — эта функция создаёт один кактус в точке с координатами x=150, y=340.

## Функция `createCactus()` — фабрика кактусов

Эта функция — как рецепт. Она создаёт новый кактус со случайными параметрами.

```javascript
function createCactus(x, y) {
  return {
    x: x,
    y: y,
    type: 'saguaro',
    height: random(120, 180),
    width: random(20, 35),
    arms: floor(random(2, 5)),
    color: color(40, 140, 40)
  };
}
```

### Что такое `return`?
Ключевое слово `return` означает "верни результат". Функция создаёт объект (набор характеристик) и отдаёт его тому, кто её вызвал.

### Что внутри кактуса?

*   **`x, y`** — координаты, где стоит кактус.
*   **`type: 'saguaro'`** — тип кактуса (сагуаро — высокий кактус с ветвями).
*   **`height: random(120, 180)`** — высота от 120 до 180 пикселей (случайное число).
*   **`width: random(20, 35)`** — ширина от 20 до 35 пикселей.
*   **`arms: floor(random(2, 5))`** — количество веток от 2 до 4 (функция `floor()` округляет вниз до целого числа).
*   **`color: color(40, 140, 40)`** — зелёный цвет (красный=40, зелёный=140, синий=40).
