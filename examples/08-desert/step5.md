# Урок 5: Иголки и тени — детали и реализм

Любая картина становится живой благодаря мелочам! Давай добавим кактусу иголки и нарисуем тени, чтобы картина выглядела объёмной.

## Рисуем иголки

Иголки — это тонкие короткие линии, которые растут в случайных направлениях. Но мы не хотим, чтобы они росли где попало — только внутри контура кактуса!

```javascript
function drawSpines(cactus) {
  stroke(160, 120, 60);
  strokeWeight(0.5);

  let spineCount = floor(15);

  for (let i = 0; i < spineCount; i++) {
    let x = random(-cactus.width / 2, cactus.width / 2);
    let y = random(-cactus.height, 0);

    // Рисуем иголку только если внутри контура кактуса
    if (abs(x) < cactus.width / 2 - 2 && y > -cactus.height + 5) {
      let angle = random(TWO_PI);
      let len = random(3, 6);
      line(x, y, x + cos(angle) * len, y + sin(angle) * len);
    }
  }
}
```

### Разбираем алгоритм:

1.  **`stroke(160, 120, 60)`** — устанавливаем коричневый цвет для линий.
2.  **`strokeWeight(0.5)`** — делаем линии очень тонкими (0.5 пикселя).
3.  **`spineCount = floor(15)`** — количество иголок (15 штук).
4.  **`x = random(...)` и `y = random(...)`** — выбираем случайную точку внутри прямоугольника кактуса.
5.  **`if (abs(x) < cactus.width / 2 - 2 && y > -cactus.height + 5)`** — важное условие! Проверяем, что точка находится внутри контура кактуса (не слишком близко к краям).
    *   **`abs(x)`** — модуль числа (всегда положительное). Проверяем расстояние от центра по горизонтали.
    *   **`&&`** — оператор "И". Оба условия должны выполняться.

### Рисуем одну иголку:

```javascript
let angle = random(TWO_PI);  // Случайный угол от 0 до 360 градусов
let len = random(3, 6);      // Случайная длина от 3 до 6 пикселей
line(x, y, x + cos(angle) * len, y + sin(angle) * len);
```

**Как это работает?**
*   **`cos(angle)`** и **`sin(angle)`** — математические функции, которые переводят угол и длину в координаты X и Y.
*   Мы прибавляем эти значения к начальной точке (x, y), чтобы получить конец иголки.

## Тени от кактусов

Тени делают картину объёмной. Наше солнце справа-вверху, значит тени падают слева-внизу.

```javascript
function drawShadows() {
  // Тени от кактусов (влево, т.к. солнце справа-вверху)
  noStroke();
  fill(0, 0, 0, 40);  // Чёрный цвет с прозрачностью 40

  for (let cactus of cacti) {
    let shadowLength = cactus.height * 0.6;
    let shadowX = cactus.x - shadowLength * 0.7;

    beginShape();
    vertex(cactus.x - cactus.width / 2, cactus.y);
    vertex(cactus.x + cactus.width / 2, cactus.y);
    vertex(shadowX + cactus.width, cactus.y + shadowLength);
    vertex(shadowX - cactus.width, cactus.y + shadowLength);
    endShape(CLOSE);
  }
}
```

### Как вычисляется тень?

1.  **`shadowLength = cactus.height * 0.6`** — длина тени равна 60% от высоты кактуса.
2.  **`shadowX = cactus.x - shadowLength * 0.7`** — тень смещена влево (вычитаем из X).
3.  **`beginShape() ... endShape(CLOSE)`** — рисуем четырёхугольник (трапецию) для тени.

### Почему прозрачность 40?

**`fill(0, 0, 0, 40)`** — четвёртое число (40) означает прозрачность. Если сделать 255 — тень будет чёрной и некрасивой. А 40 создаёт мягкую, реалистичную тень.
