# Урок 2: Рисуем небо и солнце

Любая картина начинается с фона. В пустыне это голубое небо и яркое солнце. Давай научимся рисовать градиенты и светящиеся объекты!

## Функция `draw()` — главный цикл

Вспомни: функция `draw()` выполняется снова и снова, как кадры в мультфильме. Именно здесь мы рисуем всё, что видим на экране.

```javascript
function draw() {
  // Обновляем силу ветра через шум Перлина (плавные изменения)
  windStrength = map(noise(frameCount * 0.003), 0, 1, 0.5, 2.5);

  // Небо (голубой градиент)
  drawSky();

  // Солнце
  drawSun();

  // Дальние барханы
  drawDistantDunes();

  // Основной песок
  fill(235, 200, 170);
  noStroke();
  rect(0, 290, width, 110);

  // Тени от кактусов
  drawShadows();

  // Рисуем кактусы
  for (let cactus of cacti) {
    drawCactus(cactus);
  }
}
```

### Что здесь происходит?

1.  **`windStrength = ...`** — вычисляем силу ветра (об этом будет отдельный урок!).
2.  **`drawSky()`** — вызываем функцию, которая рисует небо.
3.  **`drawSun()`** — рисуем солнце.
4.  **`drawDistantDunes()`** — рисуем дальние барханы.
5.  **`rect(...)`** — рисуем песок внизу.
6.  **`drawShadows()`** — рисуем тени.
7.  **`for (let cactus of cacti)`** — цикл, который рисует каждый кактус из массива.

## Рисуем небо: градиент

Небо — это не просто один цвет. Оно плавно переходит от голубого к оранжевому у горизонта. Это называется **градиент**.

```javascript
function drawSky() {
  // Градиент неба от голубого к светло-оранжевому
  for (let y = 0; y < height * 0.7; y++) {
    let inter = map(y, 0, height * 0.7, 0, 1);
    let c = lerpColor(color(100, 160, 230), color(255, 210, 160), inter);
    stroke(c);
    line(0, y, width, y);
  }
}
```

### Как работает градиент?

1.  **`for (let y = 0; y < height * 0.7; y++)`** — цикл, который проходит по каждой строчке экрана сверху вниз (до 70% высоты).
2.  **`inter = map(y, 0, height * 0.7, 0, 1)`** — функция `map()` переводит координату `y` в число от 0 до 1. Это нужно, чтобы понять, где мы находимся: вверху (0) или внизу (1).
3.  **`lerpColor(...)`** — волшебная функция, которая смешивает два цвета. Она берёт голубой `color(100, 160, 230)` и оранжевый `color(255, 210, 160)` и смешивает их в пропорции `inter`.
4.  **`stroke(c); line(0, y, width, y)`** — рисуем одну горизонтальную линию цветом `c`.

**Важно:** Компьютер рисует так много тонких линий, что они сливаются в плавный переход!

## Рисуем солнце с ореолом

Солнце у нас не просто круг, а с красивым светящимся ореолом вокруг.

```javascript
function drawSun() {
  // Ореол солнца
  noStroke();
  for (let i = 4; i > 0; i--) {
    fill(255, 220, 100, 25 - i * 5);
    ellipse(340, 60, 55 + i * 18, 55 + i * 18);
  }
  // Само солнце
  fill(255, 240, 120);
  ellipse(340, 60, 50, 50);
}
```

### Что такое ореол?

Ореол — это несколько полупрозрачных кругов вокруг солнца. Они создают эффект свечения.

1.  **`for (let i = 4; i > 0; i--)`** — цикл от 4 до 1. Мы рисуем 4 круга.
2.  **`fill(255, 220, 100, 25 - i * 5)`** — цвет с прозрачностью. Четвёртое число (25 - i * 5) — это прозрачность. Чем больше круг, тем он прозрачнее.
3.  **`ellipse(340, 60, 55 + i * 18, 55 + i * 18)`** — рисуем круг. Размер зависит от `i`: чем больше `i`, тем больше круг.
4.  **Последний круг** — само солнце, яркое и непрозрачное.
