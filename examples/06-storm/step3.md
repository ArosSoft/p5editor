# Часть 3: Рисование туч

Тучи рисуются с помощью трёх перекрывающихся эллипсов.  
Используем цвет `(50, 50, 70)` — тёмно-серый с оттенком синего.  
`noStroke()` убирает контур, чтобы облака выглядели мягкими.

Первый эллипс расположен слева `(100, 80)`, второй — по центру `(200, 60)`, третий — справа `(300, 80)`.  
Размеры подобраны так, чтобы тучи выглядели пушистыми и естественными.

```js
  // Рисуем тучи
  fill(50, 50, 70);
  noStroke();
  ellipse(100, 80, 150, 80);
  ellipse(200, 60, 200, 100);
  ellipse(300, 80, 150, 80);
  
```
