# Часть 4: Анимация дождя

Для каждой капли из массива `raindrops` мы увеличиваем координату `y` на её скорость.  
Если капля упала ниже нижней границы холста (`height`), мы «перемещаем» её наверх — задаём `y = 0` и случайным образом меняем `x`.  
Каждая капля рисуется как вертикальная линия.  
Цвет линий — светло-голубой с прозрачностью, толщина — 2 пикселя.

```js
// Рисуем дождь
  stroke(200, 200, 255, 150);
  strokeWeight(2);
  for (let drop of raindrops) {
    drop.y += drop.speed;
    if (drop.y > height) {
      drop.y = 0;
      drop.x = random(width);
    }
    
    line(drop.x, drop.y, drop.x, drop.y + drop.length);
  }
  
```
