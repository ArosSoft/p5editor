# Шаг 5: Звёзды, которые падают

Займёмся падающими звёздами. Они хранятся в отдельном списке `shootingStars`. На каждом кадре мы «оживляем» те звёзды, которые уже есть.

**Что происходит с каждой падающей звездой?**

1.  **Движение:** Её координаты (`x`, `y`) меняются — к ним прибавляется скорость (`vx`, `vy`). Звезда летит.
2.  **Затухание:** Её «жизненная сила» (`life`) уменьшается. Звезда постепенно угасает.
3.  **Исчезновение:** Как только `life` становится равна нулю, звезда удаляется из списка. Мы перебираем список с конца, чтобы удаление не сбило с толку наш цикл.
4.  **Рисование хвоста:** Мы рисуем короткую белую линию от нового положения звезды к её старому положению. Это создаёт эффект стремительно падающей звезды с хвостиком.

![Рисунок](images/105.PNG)
*Рисунок 4. По небу пролетают падающие звёзды, оставляя за собой светящийся след.*
```js
// Проходим по всем падающим звёздам с конца (чтобы удобно удалять)
for (let i = shootingStars.length - 1; i >= 0; i--) {
  let s = shootingStars[i];
  
  // Двигаем звезду
  s.x += s.vx;
  s.y += s.vy;
  
  // Звезда затухает
  s.life -= 2;
  
  // Если звезда погасла — удаляем её
  if (s.life <= 0) {
    shootingStars.splice(i, 1);
  }
  
  // Рисуем хвост звезды (линию)
  stroke(255, 255, 255, s.life);
  strokeWeight(2);
  line(s.x, s.y, s.x - s.vx * 2, s.y - s.vy * 2);
}
```
