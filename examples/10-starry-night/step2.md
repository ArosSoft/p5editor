# Шаг 2: Зажигаем звёзды

Теперь наполним наш список `stars` настоящими звёздами! Мы хотим, чтобы их было много и все они были разные. Для этого мы 200 раз повторяем одно и то же действие — это называется **цикл**.

**Что мы делаем в этом шаге?**

Для каждой из 200 звёзд компьютер придумывает:

*   **Координаты (`x`, `y`):** Случайное положение на холсте, чтобы звёзды не были собраны в одном углу.
*   **Размер (`size`):** От 1 до 3 пикселей. Маленькие и чуть побольше, как в настоящем небе.
*   **Начальную яркость (`brightness`):** От 100 до 255. Одни звёздочки будут изначально ярче, другие тусклее.
*   **Скорость мерцания (`twinkleSpeed`):** Это очень маленькое случайное число. Оно нужно, чтобы каждая звезда «подмигивала» в своём темпе.

Все данные о каждой звезде сохраняются в массиве `stars`.

*Рисунок 2. На небе зажглись 200 неподвижных звёзд.*
```js
// Создаём 200 звёзд с помощью цикла
  for (let i = 0; i < 200; i++) {
  stars.push({
    x: random(width),        // случайная позиция по ширине
    y: random(height),       // случайная позиция по высоте
    size: random(1, 3),      // случайный размер от 1 до 3
    brightness: random(100, 255), // случайная яркость
    twinkleSpeed: random(0.01, 0.05) // скорость мерцания
    });
  }
```
