# Шаг 6: Откуда берутся новые падающие звёзды?

Старые падающие звёзды исчезают, но на их месте должны появляться новые. Иначе небо скоро опустеет.

**Как это работает?**

Мы используем генератор случайных чисел. Представь, что мы бросаем 100-гранный кубик на каждом кадре.

*   Если выпадает единица (а это бывает нечасто, всего в 1% случаев), то мы создаём новую падающую звезду.
*   **Где она появляется?** Вверху экрана (`y = 0`).
*   **Куда летит?** Её скорость по вертикали (`vy`) всегда направлена вниз, а по горизонтали (`vx`) может быть разной — чуть влево или чуть вправо. Поэтому звёзды падают не строго вниз, а немного по диагонали.
*   **Новая звезда яркая:** Её `life` (жизненная сила) равна 255 — это максимум.

Так благодаря случайности на нашем небе то и дело вспыхивают новые падающие звёзды.
```js
// С вероятностью 1% создаём новую падающую звезду
if (random() < 0.01) {
  shootingStars.push({
    x: random(width),     // случайная позиция по ширине
    y: 0,                 // начинаем с верхнего края
    vx: random(-2, 2),    // случайная горизонтальная скорость
    vy: random(5, 10),    // скорость падения вниз
    life: 255             // полная яркость
  });
}
```
