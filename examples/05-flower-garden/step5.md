### Часть 5: Рисование стебля и бутона
После поворота мы рисуем стебель (линию вверх) и лепестки. **Цикл** внутри цветка рисует 6 лепестков по кругу. Математика `cos` и `sin` помогает расположить их равномерно.

```javascript
  // Рисуем стебель (линия от корня вверх на 150 пикселей)
  stroke(34, 139, 34);
  strokeWeight(3);
  line(0, 0, 0, -150);

  translate(0, -150); // Перемещаемся к верхушке стебля, где будет цветок
  noStroke();

  // Рисуем 6 лепестков по кругу
  for (let i = 0; i < 6; i++) {
    let angle = i * PI / 3; // 60 градусов между лепестками
    fill(flower.petalColor);
    // cos и sin помогают найти точку на окружности
    ellipse(cos(angle) * flower.size / 2, sin(angle) * flower.size / 2, 
            flower.size / 1.5, flower.size / 1.5);
  }

  // Серединка цветка
  fill(flower.centerColor);
  ellipse(0, 0, flower.size / 2, flower.size / 2);
```
> **[Место для изображения: Пошаговая схема рисования цветка: стебель, лепестки по кругу, серединка]**
