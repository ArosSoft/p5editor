### Часть 6: Добавление деталей (бабочки)
 Оживляем картину. Бабочки летают независимо от цветов. Их координаты зависят от времени (`frameCount`) и тригонометрических функций (`sin`, `cos`), чтобы они плавно порхали по кругу.

```javascript
  // Рисуем 3 бабочек
  for (let i = 0; i < 3; i++) {
    // Вычисляем координаты бабочки с учетом времени
    let x = 100 + i * 100 + sin(frameCount * 0.05 + i) * 20;
    let y = 100 + cos(frameCount * 0.03 + i) * 20;

    fill(255, 0, 255); // Розовый цвет
    // Левое крыло
    ellipse(x - 5, y, 10, 5);
    // Правое крыло
    ellipse(x + 5, y, 10, 5);
    // Тельце
    ellipse(x, y - 2, 5, 5);
  }
```
> **[Место для изображения: Крупный план бабочки с подписями: левое крыло, правое крыло, тельце]**
