# Урок 2: Создаём деревья с разными свойствами

В осеннем парке растут разные деревья: дубы, клёны и берёзы. У каждого дерева своя высота, ширина и форма кроны. Давайте создадим такие деревья в коде!

## Что такое объект?
Объект — это как анкета, у которой есть поля (свойства). Например, анкета дерева:
*   `x` — где стоит дерево
*   `y` — на какой высоте
*   `crownWidth` — ширина кроны
*   `crownHeight` — высота кроны
*   `trunkHeight` — высота ствола
*   `trunkWidth` — толщина ствола
*   `type` — тип дерева (дуб, клён, берёза)

## Создаём массив деревьев
Добавь этот код в функцию `setup()` после `createCanvas(400, 400)`:

```javascript
  // Позиции деревьев с высокими и реалистичными кронами
  treePositions = [
    {
      x: 120, y: 300,
      crownWidth: 90,
      crownHeight: 110,
      trunkHeight: 180,
      trunkWidth: 25,
      type: 'oak'
    },
    {
      x: 280, y: 300,
      crownWidth: 110,
      crownHeight: 130,
      trunkHeight: 220,
      trunkWidth: 30,
      type: 'maple'
    },
    {
      x: 200, y: 300,
      crownWidth: 80,
      crownHeight: 100,
      trunkHeight: 160,
      trunkWidth: 22,
      type: 'birch'
    }
  ];
```

## Почему так много чисел?
Каждое дерево — это объект с 7 свойствами. Мы задаём 3 дерева, поэтому код получается длинным. Но зато каждое дерево будет уникальным!

*   **Дуб (oak)** — крепкий, с широкой кроной
*   **Клён (maple)** — высокий, с большой кроной
*   **Берёза (birch)** — тонкая и изящная

## Как обращаться к свойствам?
Чтобы получить свойство объекта, используй точку:

```javascript
let firstTree = treePositions[0];
print(firstTree.x);        // 120
print(firstTree.type);     // 'oak'
print(firstTree.crownWidth); // 90
```

### Задание
Попробуй изменить первое дерево:
1.  Сделай дуб выше: измени `trunkHeight: 180` на `trunkHeight: 200`
2.  Поменяй тип дерева на `'maple'`

### Почему массив присваивается, а не заполняется циклом?
Деревьев мало (всего 3), и каждое уникальное. Проще написать их свойства вручную, чем использовать цикл. Циклы удобны, когда объектов много и они похожи.
