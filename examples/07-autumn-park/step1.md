# Урок 1: Знакомство с осенним парком

Добро пожаловать в осенний парк! Сегодня мы создадим красивую сцену с деревьями и падающими листьями. Этот проект сложнее предыдущих, но мы разберём всё по шагам!

## Что мы будем использовать
В этом проекте мы научимся:
*   **Создавать сложные объекты** — с разными свойствами
*   **Писать собственные функции** — чтобы код был аккуратным
*   **Работать с цветом HSB** — для красивых осенних оттенков
*   **Делать плавную анимацию** — с вращением и покачиванием

## Переменные для проекта
Посмотри на начало кода:

```javascript
let leaves = [];
let treePositions = [];
let grassBlades = [];
```

Здесь мы создаём три массива:
*   `leaves` — сюда будем складывать данные о каждом листике
*   `treePositions` — здесь будут храниться настройки деревьев
*   `grassBlades` — тут сохраним информацию о каждой травинке

## Функция setup()
Как всегда, `setup()` выполняется один раз при запуске:

```javascript
function setup() {
  createCanvas(400, 400);
  // ... дальше мы будем добавлять код
}
```

### Задание
Напиши этот код в редакторе и запусти программу. Ты должен увидеть пустой холст 400 на 400 пикселей.

![Пустой холст](images/empty-canvas.png)

### Почему несколько массивов?
Каждый массив хранит свой тип объектов:
*   Деревья стоят на месте — их мало (всего 3)
*   Трава растёт везде — её много (около 100 травинок)
*   Листья падают и обновляются — их 60 штук

Такое разделение делает код понятным и удобным!
