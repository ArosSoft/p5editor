# Часть 2. Функция setup() — настройка мира

## Что такое setup()?
`setup()` выполняется **один раз** при запуске программы.

> **Аналогия:** Ты один раз расставляешь краски и кисточки перед уроком рисования.

```javascript
function setup() {
  createCanvas(600, 600);
  
  planets.push(new Planet(color(200, 150, 100), 10, 80, 0.02)); // Меркурий
  planets.push(new Planet(color(255, 200, 100), 14, 120, 0.015)); // Венера
  planets.push(new Planet(color(100, 150, 255), 16, 160, 0.01)); // Земля
  planets.push(new Planet(color(255, 100, 100), 12, 200, 0.008)); // Марс
  planets.push(new Planet(color(200, 150, 100), 22, 260, 0.005)); // Юпитер
}
```

## Разбираем:

**1. Создаём холст:**
```javascript
createCanvas(600, 600);  // «Лист бумаги» 600×600 пикселей
```

**2. Добавляем планеты:**
```javascript
planets.push(new Planet(...));  // Положи новую планету в массив
```

## Параметры планеты:

| Параметр | Пример | Что означает? |
|----------|--------|---------------|
| **Цвет** | `color(200, 150, 100)` | RGB: красный, зелёный, синий (0–255) |
| **Размер** | `10` | Радиус в пикселях |
| **Расстояние** | `80` | Как далеко от Солнца |
| **Скорость** | `0.02` | Как быстро вращается |

> **Меркурий самый быстрый**, потому что ближе к Солнцу!
