# Часть 4. Рисование Солнца и орбит

## Солнце с эффектом свечения

Рисуем 4 круга: 3 полупрозрачных (свечение) + 1 яркий (Само Солнце).

```javascript
// Солнце
for (let i = 3; i > 0; i--) {
  fill(255, 200, 0, 50);
  ellipse(0, 0, 60 + i * 20, 60 + i * 20);
}
fill(255, 255, 0);
ellipse(0, 0, 50, 50);
```

| Круг | Размер | Цвет | Зачем? |
|------|--------|------|--------|
| 1–3 | 120, 100, 80 | Оранжевый, прозрачный 50 | Свечение |
| 4 | 50 | Ярко-жёлтый | Солнце |

- `fill(255, 200, 0, 50)` — оранжевый цвет, 50 = полупрозрачный
- `ellipse(0, 0, ...)` — круг с центром в (0, 0)

## Орбиты

```javascript
noFill();
stroke(100);
strokeWeight(0.5);
for (let planet of planets) {
  ellipse(0, 0, planet.distance * 2, planet.distance * 2);
}
```

| Команда | Что делает? |
|---------|-------------|
| `noFill()` | Не закрашивать (прозрачный внутри) |
| `stroke(100)` | Серая линия |
| `strokeWeight(0.5)` | Тонкая линия |

> **Почему `distance * 2`?** `distance` — это радиус, а `ellipse()` нужен диаметр.
