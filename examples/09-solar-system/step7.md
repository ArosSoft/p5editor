# Урок 7: Солнце и орбиты — центр нашей системы

В этом уроке мы нарисуем Солнце с красивым эффектом свечения и добавим орбиты для планет.

## Где рисуется Солнце?

В функции `draw()`, после `translate()` и перед циклом с планетами:

```javascript
// Рисуем Солнце с эффектом свечения
for (let i = 3; i > 0; i--) {
  fill(255, 200, 0, 50);
  ellipse(0, 0, 60 + i * 20, 60 + i * 20);
}
// Само Солнце (яркое)
fill(255, 255, 0);
ellipse(0, 0, 50, 50);

// Рисуем орбиты планет
noFill();
stroke(100);
strokeWeight(0.5);
for (let planet of planets) {
  ellipse(0, 0, planet.distance * 2, planet.distance * 2);
}
```

## Солнце с эффектом свечения

### Почему несколько кругов?

Чтобы создать эффект **свечения** (glow), мы рисуем несколько полупрозрачных кругов вокруг Солнца.

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Слои свечения Солнца]**

### Цикл для свечения

```javascript
for (let i = 3; i > 0; i--) {
  fill(255, 200, 0, 50);
  ellipse(0, 0, 60 + i * 20, 60 + i * 20);
}
```

### Как работает этот цикл?

| Итерация | i | Размер круга | Цвет |
|----------|---|--------------|------|
| 1 | 3 | 60 + 3×20 = 120 | Оранжевый, прозрачный 50 |
| 2 | 2 | 60 + 2×20 = 100 | Оранжевый, прозрачный 50 |
| 3 | 1 | 60 + 1×20 = 80 | Оранжевый, прозрачный 50 |

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Три полупрозрачных круга друг в друге]**

### Что означает `50` в `fill(255, 200, 0, 50)`?

Это **прозрачность** (alpha):
- `0` — полностью прозрачный (невидимый)
- `255` — полностью непрозрачный
- `50` — полупрозрачный (как туман)

Когда несколько полупрозрачных кругов накладываются, получается красивое свечение!

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Эффект наложения полупрозрачных слоёв]**

### Яркое Солнце

После свечения рисуем само Солнце:

```javascript
fill(255, 255, 0);       // Ярко-жёлтый (без прозрачности)
ellipse(0, 0, 50, 50);   // Диаметр 50 пикселей
```

Оно рисуется **последним**, поэтому перекрывает центр свечения.

## Орбиты планет

### Рисуем невидимые дорожки

```javascript
noFill();            // Без заливки (прозрачные внутри)
stroke(100);         // Серый цвет линий
strokeWeight(0.5);   // Тонкие линии (0.5 пикселя)

for (let planet of planets) {
  ellipse(0, 0, planet.distance * 2, planet.distance * 2);
}
```

### Почему `distance * 2`?

Потому что `distance` — это **радиус** (расстояние от центра), а `ellipse()` нужен **диаметр**!

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Радиус и диаметр орбиты]**

| Планета | distance (радиус) | diameter для ellipse() |
|---------|-------------------|------------------------|
| Меркурий | 80 | 160 |
| Венера | 120 | 240 |
| Земля | 160 | 320 |
| ... | ... | ... |

### Почему орбиты серые?

```javascript
stroke(100);  // 100 = серый цвет (от 0 до 255)
```

Орбиты должны быть едва заметны, чтобы не отвлекать от планет!

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Серые орбиты на чёрном фоне]**

## Порядок рисования важен!

Вот правильный порядок:

```javascript
// 1. Сначала свечение (задний план)
for (let i = 3; i > 0; i--) {
  // ...
}

// 2. Потом само Солнце
fill(255, 255, 0);
ellipse(0, 0, 50, 50);

// 3. Потом орбиты
noFill();
// ...

// 4. Потом планеты (передний план)
for (let planet of planets) {
  planet.update();
  planet.show();
}
```

Если поменять порядок — планеты могут оказаться **под** Солнцем или орбитами!

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Слои: фон → свечение → Солнце → орбиты → планеты]**

## Команды для рисования

### Краткая справка:

| Команда | Что делает | Пример |
|---------|------------|--------|
| `fill(color)` | Цвет заливки | `fill(255, 0, 0)` — красный |
| `noFill()` | Без заливки | Прозрачный внутри |
| `stroke(color)` | Цвет обводки | `stroke(100)` — серый |
| `noStroke()` | Без обводки | Нет линии вокруг |
| `strokeWeight(n)` | Толщина линии | `strokeWeight(3)` — 3 пикселя |
| `ellipse(x, y, w, h)` | Круг/овал | `ellipse(0, 0, 50, 50)` |
