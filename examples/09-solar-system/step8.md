## 1. Кольца Сатурна

### Как добавить кольца?

Уже в коде есть возможность рисовать кольца! Нужно передать `true` при создании планеты:

```javascript
planets.push(new Planet(
  color(210, 180, 140),  // Цвет
  20,                     // Размер
  320,                    // Расстояние
  0.004,                  // Скорость
  true,                   // ✅ Есть кольца!
  "Сатурн"                // Название
));
```

### Как это работает?

В методе `show()` есть проверка:

```javascript
if (this.hasRings) {
  noFill();
  stroke(200, 180, 150, 150);  // Бежевый, полупрозрачный
  strokeWeight(3);
  ellipse(x, y, this.radius * 2.5, this.radius * 2.5);
}
```

**Важно:** Кольца рисуются **ДО** планеты, чтобы быть на заднем плане!

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Сатурн с кольцами]**

### Хочешь добавить кольца другой планете?

Просто измени `false` на `true` при создании:

```javascript
// Земля с кольцами (в реальности их нет, но кто мешает?)
planets.push(new Planet(..., true, "Земля"));
```

## 2. Подписи планет

### Как это работает?

В методе `show()` уже есть код для рисования названий:

```javascript
// Рисуем название планеты
fill(255);              // Белый цвет
textSize(10);           // Размер 10 пикселей
textAlign(CENTER);      // По центру
text(this.name, x, y - this.radius - 5);  // Над планетой
```

### Что такое `text()`?

Команда `text("привет", x, y)` рисует текст в координатах (x, y).

**Пример:**
```javascript
text("Земля", 100, 200);  // Написать "Земля" в точке (100, 200)
```

### Почему `y - this.radius - 5`?

Чтобы текст был **над** планетой:
- `y` — центр планеты
- `- this.radius` — поднимаемся на радиус вверх
- `- 5` — ещё 5 пикселей отступа

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Текст над планетой с отступом]**

### Как изменить размер текста?

```javascript
textSize(15);  // Больше текст (было 10)
```

## 3. Идеи для улучшения

### 🌙 Луна для Земли

Добавь маленький спутник вокруг Земли!

**В методе `show()`**, после рисования планеты:

```javascript
// Если это Земля, рисуем Луну
if (this.name === "Земля") {
  let moonAngle = this.angle * 12;  // Луна быстрее Земли
  let moonX = x + cos(moonAngle) * 25;
  let moonY = y + sin(moonAngle) * 25;
  
  fill(200);
  ellipse(moonX, moonY, 4, 4);  // Маленькая Луна
}
```

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Земля с Луной]**

### ☄️ Астероидный пояс

Добавь много мелких точек между Марсом и Юпитером!

**В `draw()`,** после рисования орбит:

```javascript
// Астероидный пояс
stroke(150);
strokeWeight(1);
for (let i = 0; i < 100; i++) {
  let angle = i * 0.1 + frameCount * 0.001;  // Медленное движение
  let distance = 230 + random(-10, 10);      // Пояс вокруг 230
  let ax = cos(angle) * distance;
  let ay = sin(angle) * distance;
  point(ax, ay);  // Точка-астероид
}
```

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Астероидный пояс]**

### ⭐ Звёзды на фоне

Добавь случайные звёзды на чёрный фон!

**В начале `draw()`,** после `background(0)`:

```javascript
// Звёзды на фоне
stroke(255);
strokeWeight(1);
for (let i = 0; i < 50; i++) {
  // Используем fixed random для постоянства
  let sx = random(0, width);
  let sy = random(0, height);
  point(sx, sy);
}
```

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Звёздное небо]**

### 🎨 Цветные орбиты

Сделай каждую орбиту своего цвета!

**Вместо серых орбит:**

```javascript
for (let planet of planets) {
  stroke(planet.color);      // Цвет орбиты = цвету планеты
  strokeWeight(1);
  noFill();
  ellipse(0, 0, planet.distance * 2, planet.distance * 2);
}
```

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Разноцветные орбиты]**

### 🚀 Запуск кометы

Добавь объект с очень вытянутой орбитой!

**В `setup()`:**
```javascript
// Комета с эллиптической орбитой
let comet = {
  angle: 0,
  speed: 0.03,
  distanceMin: 50,   // Ближайшая точка
  distanceMax: 500   // Дальняя точка
};
```

**В `draw()`:**
```javascript
// Комета движется по эллипсу
comet.angle += comet.speed;
let cometDist = comet.distanceMin + 
                (comet.distanceMax - comet.distanceMin) * 
                (sin(comet.angle) + 1) / 2;
let cx = cos(comet.angle) * cometDist;
let cy = sin(comet.angle) * cometDist;

fill(255);
ellipse(cx, cy, 6, 6);
```

**[ЗДЕСЬ БУДЕТ ИЗОБРАЖЕНИЕ: Комета на эллиптической орбите]**
