# Шаг 3: Цикл for — таблица умножения

## Что такое циклы?

**Циклы** позволяют выполнять один и тот же код многократно. Это избавляет от необходимости писать одинаковые строки снова и снова.

## Синтаксис цикла for

```javascript
for (инициализация; условие; шаг) {
  // Код, который выполняется в каждом цикле
}
```

## Пример: таблица умножения на 5

```javascript
for (let i = 1; i <= 10; i++) {
  console.log(`5 × ${i} = ${5 * i}`);
}
```

## Как это работает?

1. **Инициализация** (`let i = 1`) — выполняется один раз перед началом цикла
2. **Условие** (`i <= 10`) — проверяется перед каждой итерацией
3. **Тело цикла** — выполняется, если условие истинно
4. **Шаг** (`i++`) — выполняется после каждой итерации

## Вывод таблицы умножения

```
5 × 1 = 5
5 × 2 = 10
5 × 3 = 15
5 × 4 = 20
5 × 5 = 25
5 × 6 = 30
5 × 7 = 35
5 × 8 = 40
5 × 9 = 45
5 × 10 = 50
```

## Разные варианты цикла for

```javascript
// От 0 до 9
for (let i = 0; i < 10; i++) {
  console.log(i);
}

// От 10 до 1 (обратный порядок)
for (let i = 10; i >= 1; i--) {
  console.log(i);
}

// С шагом 2 (только чётные)
for (let i = 0; i <= 20; i += 2) {
  console.log(i);
}
```
