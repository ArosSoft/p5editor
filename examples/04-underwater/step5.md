
# Шаг 5: Создаём пузырьки воздуха ✨

Добавим пузырьки, которые появляются со дна и поднимаются вверх.

## Код
```javascript
if (random() < 0.1) {
  bubbles.push({
    x: random(width),
    y: height,
    size: random(5, 15),
    speed: random(1, 3)
  });
}
```

## Объяснение
`if (random() < 0.1)` — 10% вероятность каждый кадр.

`bubbles.push({...})` — добавляет новый пузырёк в массив.

`x: random(width)` — случайная позиция по ширине.

`y: height` — начинают снизу (y = 400).

`size: random(5, 15)` — случайный размер.

`speed: random(1, 3)` — случайная скорость.
