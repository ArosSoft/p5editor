# Шаг 1: Создаём контейнеры для рыбок и пузырьков 🗂️

Прежде чем рисовать рыбок и пузырьки, нам нужно место, где мы будем их хранить. В программировании для этого используют **массивы**.

## Код
```javascript
let fish = [];
let bubbles = [];
```

## Объяснение
`let` — это команда, которая говорит: "Создай новую переменную".

`fish` — контейнер (массив) для хранения рыбок.

`bubbles` — контейнер (массив) для хранения пузырьков.

Квадратные скобки `[]` означают, что это массив (список).

Пока наши коробки пустые, но скоро мы их наполним!

> 🖼️ **[Здесь будет картинка: две пустые коробки с подписями "Рыбки" и "Пузырьки"]**
