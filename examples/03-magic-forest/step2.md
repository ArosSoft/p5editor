# Урок 2: Создаём деревья с помощью цикла

Теперь наполним наш лес деревьями. Но рисовать каждое дерево вручную — скучно и долго. Давайте используем **цикл**!

## Что такое цикл?
Цикл — это способ заставить компьютер выполнить одно и то же действие много раз. В JavaScript есть цикл `for`, который выглядит так:

```javascript
for (let i = 0; i < 10; i++) {
  // Код, который повторится 10 раз
}
```

Давай разберём, что здесь происходит:
1.  `let i = 0` — создаём переменную `i` (счётчик) и начинаем с нуля
2.  `i < 10` — продолжаем, пока `i` меньше 10
3.  `i++` — после каждого повторения увеличиваем `i` на 1

## Случайные числа
Чтобы деревья были разными, используем функцию `random()`. Она возвращает случайное число.

*   `random(100)` — случайное число от 0 до 100
*   `random(50, 100)` — случайное число от 50 до 100

## Создаём массив деревьев
Добавь этот код в функцию `setup()` после `createCanvas(400, 400)`:

```javascript
  // Создаём деревья
  for (let i = 0; i < 10; i++) {
    trees.push({
      x: random(width),
      y: height - random(50, 100),
      height: random(80, 150),
      width: random(20, 40)
    });
  }
```

## Что такое `push`?
`push()` — это команда, которая добавляет новый элемент в конец массива. Как будто кладёшь новую карточку в коробку.

## Что хранит каждое дерево?
Мы создаём **объект** — это набор данных, у которого есть имена (свойства):
*   `x` — случайная позиция по горизонтали (от 0 до ширины холста)
*   `y` — позиция по вертикали (внизу холста, но с небольшим разбросом)
*   `height` — высота дерева (от 80 до 150 пикселей)
*   `width` — ширина ствола (от 20 до 40 пикселей)
