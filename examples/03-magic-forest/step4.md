# Урок 4: Рисуем ночное небо и траву

Теперь самое время нарисовать фон нашего волшебного леса. Начнём с неба и травы.

## Функция draw()
Всё, что мы нарисуем в `draw()`, будет появляться на экране. Эта функция выполняется много раз в секунду (обычно 60 раз!).

## Рисуем градиентное небо
Ночное небо не бывает одного цвета. Внизу оно темнее, вверху — светлее. Чтобы сделать такой эффект, нарисуем много горизонтальных линий разного цвета.

Добавь этот код в функцию `draw()`:

```javascript
function draw() {
  // Небо (градиент)
  for (let y = 0; y < height; y++) {
    let dark = map(y, 0, height, 50, 0);
    stroke(25, 25, 50 + dark);
    line(0, y, width, y);
  }
}
```

## Что делает `map`?
Функция `map()` — как переводчик между числами. Она берёт число из одного диапазона и переводит в другой.

```javascript
map(y, 0, height, 50, 0)
```

Это значит:
*   Берём `y` (текущую строку)
*   Если `y = 0` (верх холста) → получаем 50
*   Если `y = height` (низ холста) → получаем 0
*   Для всех остальных `y` — вычисляем промежуточное значение

Так мы создаём плавный переход от тёмного к светлому!

## Рисуем траву
Трава будет простым зелёным прямоугольником внизу холста:

```javascript
  // Рисуем траву
  fill(20, 80, 20);  // Тёмно-зелёный цвет
  noStroke();        // Убираем обводку
  rect(0, height - 50, width, 50);
```

## Как работает `rect`?
`rect(x, y, ширина, высота)` рисует прямоугольник. Но помни: `x` и `y` — это **верхний левый угол**!

*   `0` — начинаем от левого края
*   `height - 50` — отступаем 50 пикселей от низа
*   `width` — ширина на весь холст
*   `50` — высота травы 50 пикселей

### Почему градиент рисуется циклом?
Мы рисуем 400 линий (по одной на каждый пиксель высоты). Каждая линия — на 1 пиксель ниже предыдущей и чуть темнее. Это как раскрасить страницу карандашом, накладывая штрихи!
