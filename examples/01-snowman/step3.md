# Урок 3: Лицо и пуговицы

Снеговик готов, но ему чего-то не хватает. Давай добавим ему лицо и пуговицы. Для глаз и пуговиц снова используем `ellipse`, а для носа — новую фигуру `triangle`.

## Глаза и пуговицы
Чтобы нарисовать глаза, нужно поставить две черные точки на голове.

```javascript
  // Глаза (черные кружки)
  fill(0); // Черный цвет
  ellipse(185, 105, 8, 8); // Левый глаз
  ellipse(215, 105, 8, 8); // Правый глаз

  // Пуговицы
  ellipse(200, 170, 8, 8);
  ellipse(200, 200, 8, 8);
  ellipse(200, 230, 8, 8);
```

## Нос-морковка (треугольник)
Треугольник рисуется по трем точкам: `triangle(x1, y1, x2, y2, x3, y3)`.

```javascript
  // Нос (оранжевый треугольник)
  fill(255, 150, 0); // Оранжевый
  triangle(200, 115,  // Вершина у основания носа
           230, 115,  // Правый угол
           200, 130); // Нижний угол (кончик носа)
```

## Улыбка (дуга)
Для улыбки используем команду `arc`, которая рисует часть овала.

```javascript
  // Улыбка
  noFill();        // Без заливки
  stroke(0);       // Черная обводка
  strokeWeight(2); // Толщина линии 2 пикселя
  arc(200, 125, 30, 15, 0, PI); // Рисуем полукруг (от 0 до PI радиан)
```
