# Урок 5: Главная задача — рисуем пальцы!

Снеговик почти готов. Посмотри на задание: *"Добавь пальцы"*. У наших рук-палок пальцев нет. Давай это исправим!

## Анализируем
Руки — это просто линии. Чтобы нарисовать пальцы, нам нужно добавить к концам этих линий маленькие ответвления (как веточки на дереве).

### Рисуем пальцы на левой руке
Представим, что левая рука — это ствол, а пальцы — это маленькие веточки, отходящие в стороны.

Добавь этот код **после** того, как нарисованы основные линии рук. Не забудь снова включить `stroke` и `strokeWeight`.

```javascript
  // Палец 1 (на левой руке, вверх)
  line(300, 140, 303, 117);
  // Палец 2 (на левой руке, вниз)
  line(300, 140, 320, 140);
  // Палец 1 (на правой руке, вверх)
  line(100, 140, 90, 120);
  // Палец 2 (на правой руке, вниз)
  line(100, 140, 75, 140);
```
