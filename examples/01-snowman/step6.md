# Урок 6: Оживляем снеговика (Анимация)

Самое интересное в программировании — это анимация. Давай заставим нашего снеговика махать нам рукой!

## Как работает анимация
Вспомни, что функция `draw()` выполняется снова и снова. Если мы будем понемногу менять координаты руки в каждом кадре, нам покажется, что она движется.

## Добавляем переменную
Переменная — это контейнер для хранения числа. Мы создадим переменную `waveOffset`, которая будет меняться.

1.  **В самом верху**, до функции `setup()`, добавь:
    ```javascript
    let waveOffset = 0;
    let waveDirection = 1;
    ```

2.  **В функции `draw()`**, перед рисованием рук, добавим логику движения:
    ```javascript
    // Анимация для руки
    waveOffset = waveOffset + waveDirection;
    if (waveOffset > 20 || waveOffset < -20) {
      waveDirection = waveDirection * -1; // Меняем направление
    }
    ```

3.  **Теперь изменим код правой руки**. Вместо фиксированной точки (300, 140) подставим нашу переменную.
    *Было:* `line(240, 170, 300, 140);`
    *Стало:*
    ```javascript
    line(240, 170, 300 + waveOffset, 140 + waveOffset/2);
    ```
