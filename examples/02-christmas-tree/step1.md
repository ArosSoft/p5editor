# Шаг 1: Создаём холст и рисуем ёлку 🎄

## Что мы будем делать

На этом уроке мы нарисуем красивую новогоднюю ёлку с помощью программирования! Ты узнаешь, как создавать холст и рисовать простые фигуры.

## Начинаем работу

### 1. Создаём холст

Первая команда — `setup()`. Это как подготовка рабочего места перед рисованием:

```javascript
function setup() {
  createCanvas(400, 400);
}
```

**`createCanvas(400, 400)`** — создаёт холст размером 400×400 пикселей. Это наш «лист бумаги» для рисования.


### 2. Делаем красивый фон

Зальём холст тёмно-синим цветом, как ночное зимнее небо:

```javascript
background(25, 25, 70);
```

Команда `background()` закрашивает весь холст одним цветом. Числа в скобках — это количество красного, зелёного и синего цвета (от 0 до 255).


### 3. Рисуем снежную полянку

Под ёлкой будет белый снег:

```javascript
fill(255);        // Белый цвет
noStroke();       // Без контура
ellipse(200, 333, 1000, 200);  // Рисуем овал
```

**`fill()`** — выбирает цвет для закрашивания фигур.  
**`noStroke()`** — убирает контур у фигуры.  
**`ellipse()`** — рисует овал (или круг, если ширина и высота равны).
