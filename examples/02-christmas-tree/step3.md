# Шаг 3: Добавляем звезду и игрушки ⭐

## Украшаем ёлку

Настоящая новогодняя ёлка не может обойтись без красивой звезды на верхушке!

## Рубиновая звезда

Звезда будет красной с золотым контуром — как настоящий рубин!

```javascript
// Настраиваем цвета
fill(200, 0, 0);      // Красный цвет звезды
stroke(255, 215, 0);  // Золотой контур
strokeWeight(2);      // Толщина контура
```

### Рисуем пятиконечную звезду

Звезда — сложная фигура, поэтому используем специальный код:

```javascript
push();               // Сохраняем текущие настройки
translate(200, 30);   // Перемещаем «карандаш» в точку (200, 30)
beginShape();         // Начинаем рисовать сложную фигуру

for (let i = 0; i < 5; i++) {
  // Вычисляем координаты вершин звезды
  let angle = TWO_PI * i / 5 - HALF_PI;
  let x = cos(angle) * 15;
  let y = sin(angle) * 15;
  vertex(x, y);

  let innerAngle = angle + TWO_PI / 10;
  let innerX = cos(innerAngle) * 7;
  let innerY = sin(innerAngle) * 7;
  vertex(innerX, innerY);
}

endShape(CLOSE);      // Заканчиваем рисовать и замыкаем фигуру
pop();                // Восстанавливаем настройки
```

**Не переживай, если этот код кажется сложным!** Главное — понять, что:
- `push()` и `pop()` — как закладка в книге, сохраняют настройки
- `translate()` — перемещает точку отсчёта
- Цикл `for` — повторяет действия 5 раз (по числу лучей звезды)
