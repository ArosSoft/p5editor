## ❌ Чего НЕ ХВАТАЕТ (рекомендуемые дополнения)

| № | Раздел | Темы | Почему важно | Пример для добавления |
|---|--------|------|--------------|----------------------|
| 1 | **Ввод-вывод данных** | `prompt()`, `alert()`, `console.log()`, ввод с клавиатуры (`keyPressed`, `keyCode`) | Базовое взаимодействие с пользователем | Калькулятор, текстовый квест |
| 2 | **Строки и текст** | Методы строк, конкатенация, шаблонные строки | Работа с текстовыми данными | Генератор историй, шифратор |
| 3 | **Отладка** | `debugger`, console.trace, breakpoints | Критический навык разработки | Специальный пример с ошибками для исправления |
| 4 | **Обработка ошибок** | try/catch, валидация данных | Надёжность кода | Пример с обработкой некорректного ввода |
| 5 | **Локальное хранилище** | localStorage, sessionStorage | Сохранение данных между сессиями | Сохранение рекорда в игре |
| 6 | **Работа с датой и временем** | Date, setTimeout, setInterval | Планирование, таймеры | Таймер обратного отсчёта, будильник |
| 7 | **JSON и данные** | JSON.parse, JSON.stringify, fetch API | Работа с внешними данными | Загрузка погоды из API |
| 8 | **Модульность** | import/export, разделение на файлы | Организация крупных проектов | Рефакторинг большого примера на модули |
| 9 | **События DOM** | createButton, createInput, changed(), mouseReleased | Создание интерфейсов | Интерактивная форма, редактор |
| 10 | **Матрицы и 2D массивы** | Двумерные массивы, вложенные циклы | Игровые поля, карты | 006-life есть, но нужен отдельный базовый пример |
| 11 | **Сортировки и поиск** | bubble sort, binary search | Фундаментальные алгоритмы | Визуализация сортировок |
| 12 | **Физика и коллизии** | Обнаружение столкновений, гравитация, отскоки | Игры, симуляции | 12-rocket-landing частично, но нужен простой пример |
| 13 | **Path и drawing** | beginShape, vertex, кривые Безье | Сложная графика | Рисование кривых, логотипы |
| 14 | **Масштабирование** | scale(), responsive canvas | Адаптивность | Пример с resize canvas |
| 15 | **Видео и камера** | createCapture, VIDEO | Работа с медиа | Детектор движения, зеркало |
| 16 | **Экспорт и сохранение** | saveCanvas, saveFrames | Сохранение результатов | Кнопка "сохранить рисунок" |
| 17 | **Тестирование** | Unit-тесты, assert | Качество кода | Пример с простыми тестами функций |
| 18 | **Git и версионирование** | Основы Git | Командная разработка | Не код, но важно добавить в программу |

## 📋 Рекомендуемая учебная программа по классам

| Тема | Что изучать | Примеры из проекта |

| Условные операторы | if/else, сравнения | 02-christmas-tree |
